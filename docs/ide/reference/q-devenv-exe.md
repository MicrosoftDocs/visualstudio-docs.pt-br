# <a name="-devenvexe"></a>/? (devenv.exe)

Exibe uma caixa de mensagem listando todas as opções de `devenv`, com uma breve descrição de cada uma delas.

## <a name="syntax"></a>Sintaxe

```shell
devenv /?
```
