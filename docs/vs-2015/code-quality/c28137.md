# <a name="c28137"></a>C28137
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28137: O argumento variável deve ser uma constante (literal)  
  
 Esse aviso é relatado quando uma chamada de função está faltando uma constante (literal) necessária. Consulte a documentação para a função.  
  
## <a name="example"></a>Exemplo  
 Por exemplo, o [ExAcquireResourceExclusiveLite](https://msdn.microsoft.com/library/windows/hardware/ff544351.aspx) rotina requer um valor de TRUE ou FALSE para o `Wait` parâmetro. O exemplo de código a seguir gera este aviso:  
  
```  
ExAcquireResourceExclusiveLite(Resource, Wait);  
```  
  
 O exemplo de código a seguir evita este aviso:  
  
```  
ExAcquireResourceExclusiveLite(Resource, TRUE);  
```
