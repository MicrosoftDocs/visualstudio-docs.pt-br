# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28230: O tipo de parâmetro não tem membro.  
  
 Este aviso indica que um argumento para uma anotação tenta acessar um `struct`, `class`, ou `union` e o membro nomeado não existe.
