# <a name="sharepoint-development-samples"></a>Exemplos de desenvolvimento do SharePoint
  Você pode usar o Visual Studio para baixar e instalar exemplos de aplicativos do SharePoint na [MSDN Code Gallery](http://go.microsoft.com/fwlink/?LinkId=254185).  
  
 Você pode baixar cada exemplo individualmente. Você também pode baixar um pacote de exemplo que tem amostras que compartilham uma tecnologia ou artigo relacionadas. Você receberá uma notificação quando as alterações no código-fonte forem publicadas para qualquer exemplo baixado.
