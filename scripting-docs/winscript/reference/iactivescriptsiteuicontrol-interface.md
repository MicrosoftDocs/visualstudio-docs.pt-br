# <a name="iactivescriptsiteuicontrol-interface"></a>Interface IActiveScriptSiteUIControl
Fornece um método para obter o comportamento de um controle de interface do usuário.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[IActiveScriptSiteUIControl::GetUIBehavior Method](../../winscript/reference/iactivescriptsiteuicontrol-getuibehavior-method.md)|Obtém um [enumeração SCRIPTUICHANDLING](../../winscript/reference/scriptuichandling-enumeration.md) que representa a maneira que um controle de interface do usuário deve ser tratado.|
