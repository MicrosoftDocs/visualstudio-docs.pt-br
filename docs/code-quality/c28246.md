# <a name="c28246"></a>C28246
C28246 de aviso: A anotação para a função ' \<name > '-Parameter ' \<parameter > ' não corresponde ao tipo do parâmetro

 Um operador __deref foi aplicado a um tipo que não é de ponteiro ao criar uma anotação.
