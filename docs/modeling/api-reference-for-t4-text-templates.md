# <a name="api-reference-for-t4-text-templates"></a>Referência de API para modelos de texto T4

A API de modelagem de texto permite invocar e personalizar a transformação de [modelos de texto](../modeling/code-generation-and-t4-text-templates.md).

## <a name="namespaces"></a>Namespaces

|Namespace|Finalidade|
|-|-|
|<xref:Microsoft.VisualStudio.TextTemplating?displayProperty=fullName>|Contém classes para a funcionalidade de transformação de modelo de texto. O mecanismo de transformação de modelo de texto é integrado ao Visual Studio e transforma arquivos de modelo de texto em arquivos de saída de texto gerados.|
|[Microsoft. VisualStudio. TextTemplating. Modeling](/previous-versions/ee844312(v=vs.140))|Fornece recursos de transformação de texto relacionados a modelos UML e linguagens específicas de domínio, como acesso a Visual Studio ModelBus.|
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost?displayProperty=fullName>|Fornece acesso ao serviço de modelagem de texto no Visual Studio.|
