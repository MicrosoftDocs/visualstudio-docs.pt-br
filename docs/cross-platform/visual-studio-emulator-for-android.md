# <a name="visual-studio-emulator-for-android"></a>Emulador do Visual Studio para Android

O Visual Studio Emulator para Android é um aplicativo de desktop para emular um dispositivo Android. É preterido no Visual Studio 2015 e não é suportado no Visual Studio 2017 ou Visual Studio 2019. Recomendamos que você use os emuladores do Google Android. Para ver o Emulador visual studio para instruções do Android para o Visual Studio 2015, escolha o Visual Studio 2015 no seletor de versão de documentação.

> [!IMPORTANT]
> Na maioria dos cenários, o emulador do Google Android é recomendado para uso, em vez do Emulador do Visual Studio para Android:
> - Visual Studio Emulator para Android não é suportado após o Visual Studio 2015.
> - As imagens do emulador mais tarde que a versão 6.0 do Android não estão disponíveis para o Visual Studio Emulator para Android.
> - O Google Android Emulator agora é compatível com [Hyper-V](/xamarin/android/get-started/installation/android-emulator/hardware-acceleration#accelerating-with-hyper-v).
> - Tanto c++ para desenvolvimento de dispositivos móveis e ferramentas de estúdio visual para Apache Cordova trabalham com o Google Android Emulador. Para saber mais, confira [Executar seu aplicativo Apache Cordova no Android](/visualstudio/cross-platform/tools-for-cordova/run-your-app/run-app-android#google-android-emulator). (Você não precisa mais desativar o Hyper-V como explicado neste artigo).
>
> Para obter mais informações sobre como configurar e usar o emulador do Google Android, consulte a [configuração do Android Emulator](/xamarin/android/get-started/installation/android-emulator/).
