# <a name="use-pattern-matching"></a>Usar correspondência de padrões

Esta refatoração aplica-se a:

- C#

**O que:** Use correspondência de padrões.

**Quando:** Você deseja usar os novos recursos de [correspondência de padrão C#](https://docs.microsoft.com/dotnet/csharp/whats-new/csharp-9#pattern-matching-enhancements) .

**Por que:** Essa refatoração tornará seu código mais claro e conciso.

## <a name="how-to"></a>Como fazer

1. Coloque o cursor dentro da instrução.

2. Pressione **Ctrl** + **.** para acionar o menu **Ações e Refatorações Rápidas**.

3. Selecione **Usar a correspondência de padrões**.
