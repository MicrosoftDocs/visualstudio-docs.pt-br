# <a name="c26495-member_uninit"></a>C26495 MEMBER_UNINIT

A variável '% Variable% ' não foi inicializada. Sempre inicialize uma variável de membro. Ver [Type.6 de diretrizes principais do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
