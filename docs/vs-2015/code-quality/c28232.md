# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28232: \_ pre \_ , \_ post \_ ou \_ Deref \_ não aplicado a nenhuma anotação  
  
 Esse aviso indica que um `_Pre_` `_Post_` operador,, ou `_Deref_` aparece em uma expressão de anotação sem uma anotação funcional subsequente; o modificador foi ignorado, mas isso indica uma anotação codificada incorretamente.
