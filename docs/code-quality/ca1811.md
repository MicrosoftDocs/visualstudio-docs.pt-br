# <a name="ca1811-avoid-uncalled-private-code"></a>CA1811: evitar código privado não chamado

|||
|-|-|
|TypeName|AvoidUncalledPrivateCode|
|CheckId|CA1811|
|Categoria|Microsoft.Performance|
|Mudança de ruptura|Não quebrando|

## <a name="cause"></a>Causa
Um membro privado ou interno (nível de montagem) não tem chamadores na assembléia, não é invocado pelo tempo de execução da linguagem comum e não é invocado por um delegado. Os seguintes membros não são verificados por esta regra:

- Membros explícitos da interface.

- Construtores estáticos.

- Construtores serializadores.

- Métodos marcados com <xref:System.Runtime.InteropServices.ComRegisterFunctionAttribute?displayProperty=fullName> ou <xref:System.Runtime.InteropServices.ComUnregisterFunctionAttribute?displayProperty=fullName>.

- Membros que são sobre-substituições.

## <a name="rule-description"></a>Descrição da regra
Essa regra pode reportar falsos positivos se ocorrerem pontos de entrada que não sejam identificados atualmente pela lógica da regra. Além disso, um compilador pode emitir código não calável em um conjunto.

## <a name="how-to-fix-violations"></a>Como corrigir violações
Para corrigir uma violação desta regra, remova o código não callable ou adicione o código que o chama.

## <a name="when-to-suppress-warnings"></a>Quando suprimir avisos
É seguro suprimir um aviso desta regra. Para isso, consulte [Suprimir os avisos de análise de código](../code-quality/in-source-suppression-overview.md).
