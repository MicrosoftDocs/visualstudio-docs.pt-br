# <a name="c26463-useconstforelements"></a>C26463 USE_CONST_FOR_ELEMENTS
  Os elementos de matriz '% % de matriz' são atribuídos somente uma vez, marque-os `const`. Ver [con.4 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
