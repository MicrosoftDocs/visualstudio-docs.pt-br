# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C6995: falha ao salvar o arquivo de log XML  
  
 Esse aviso indica que a ferramenta de análise de código não pode criar o log de defeito, que é a saída da análise de código.  
  
 Esse erro pode indicar um erro de disco ou indicar que você não tem permissão para criar um arquivo no diretório especificado.
