# <a name="ca0505"></a>CA0505

A propriedade preterida será ignorada porque a propriedade superceding está definida.

Aviso CA505 é gerado quando tanto preterido especificado e substituindo propriedades de análise de código contêm valores em um projeto ou criar arquivo de configuração. Para resolver esse problema, remova a propriedade preterida.
