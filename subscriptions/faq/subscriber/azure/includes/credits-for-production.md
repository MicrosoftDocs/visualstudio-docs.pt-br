## <a name="can-i-use-my-monthly-azure-credits-to-run-production-applications"></a>Posso usar meus créditos Azure mensais para executar aplicativos de produção?

Os créditos Azure mensais para assinantes do Visual Studio destinam-se especificamente ao uso individual de desenvolvimento/teste. Para executar cargas de trabalho de produção, você precisará atualizar para uma assinatura paga do Azure, como o preço [pago conforme o uso](https://azure.microsoft.com/offers/ms-azr-0003p/).
