# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28220: Expressão inteira esperada para anotação do parâmetro na anotação  
  
 Este aviso indica que um parâmetro em uma anotação deve ser uma expressão de inteiro, e algum outro tipo foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente.
