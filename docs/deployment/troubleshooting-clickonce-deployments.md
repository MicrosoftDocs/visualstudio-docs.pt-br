# <a name="troubleshoot-clickonce-deployments"></a>Solucionar problemas de implantações do ClickOnce
Este tópico ajuda você a diagnosticar e resolver os problemas mais comuns com [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] implantações.  
  
 Na maioria dos casos, um [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] aplicativo será baixado para o computador de um usuário e executado sem problemas. Há alguns casos, no entanto, em que os problemas de configuração de servidor ou aplicativo Web podem causar problemas imprevistos.
