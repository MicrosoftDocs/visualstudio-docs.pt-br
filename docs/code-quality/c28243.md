# <a name="c28243"></a>C28243
aviso C28243: a anotação para a função no parâmetro requer mais desreferências do que o tipo real anotado permite

 O número de operadores `__deref` em uma anotação é maior que o número de níveis de ponteiro definidos pelo tipo de parâmetro. Corrija esse aviso alterando o número na anotação ou os níveis de ponteiro do parâmetro referenciado.
