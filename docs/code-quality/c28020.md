# <a name="c28020"></a>C28020
warning C28020: A expressão \<expr > não é verdadeira nesta chamada

 Esse aviso é relatado quando os \_atende às\_ expressão listado não é verdadeira. Frequentemente, isso indica um parâmetro incorreto.

 Se isso ocorrer em uma declaração de função, as anotações indicam uma condição de impossível.
