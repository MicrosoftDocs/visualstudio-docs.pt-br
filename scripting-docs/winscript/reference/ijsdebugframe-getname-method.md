# <a name="ijsdebugframegetname-method"></a>Método IJsDebugFrame::GetName
Obtém o nome amigável do quadro de pilha.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pName`  
 [out] O nome do quadro de pilha.  
  
## <a name="return-value"></a>Valor de retorno  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jscript9diag.h
