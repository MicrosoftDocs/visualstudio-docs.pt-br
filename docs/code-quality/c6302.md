# <a name="c6302"></a>C6302
aviso C6302: cadeia de caracteres de formato incompatível: a cadeia de caracteres passou como parâmetro \<number > quando uma cadeia de caracteres larga é necessária na chamada de \<Function >

 Esse aviso indica que a cadeia de caracteres de formato especifica que uma cadeia de caracteres larga é necessária. No entanto, uma cadeia de caracteres está sendo passada. Esse defeito provavelmente causará uma falha ou uma corrupção de alguma forma.

## <a name="example"></a>Exemplo
 O código de exemplo a seguir gera esse aviso porque uma cadeia de caracteres é passada para a função `wprintf`:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 O código de exemplo a seguir usa `%hs` para especificar uma cadeia de caracteres de byte único com a função `wprintf`:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 O código de exemplo a seguir usa a função de manipulação de cadeia de caracteres segura `wprintf_s` para corrigir este aviso:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
