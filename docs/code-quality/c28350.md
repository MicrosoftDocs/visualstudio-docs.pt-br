# <a name="c28350"></a>C28350
warning C28350: A anotação \<anotação > descreve uma situação que não é aplicável condicionalmente.

 Normalmente, esse aviso é gerado quando uma anotação é aplicada em que o tipo de C/C++ está sendo inspecionado.
