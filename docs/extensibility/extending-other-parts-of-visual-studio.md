# <a name="extend-other-parts-of-visual-studio"></a>Estender outras partes do Visual Studio

Há muito mais partes da IU do Visual Studio que você pode estender. Aqui nós mostramos apenas alguns.

## <a name="create-a-vspackage"></a>Criar um VSPackage

Os blocos básicos de construção do Visual Studio Extensibility são VSPackages.  Saiba como adicionar um VSPackage: [Crie uma extensão com um VSPackage](../extensibility/creating-an-extension-with-a-vspackage.md)

## <a name="extend-the-toolbox"></a>Estender a caixa de ferramentas

Saiba como adicionar novos controles e outros itens à caixa de ferramentas e como usar a funcionalidade do Toolbox:

- [Criar um controle de caixa de ferramentas WPF](../extensibility/creating-a-wpf-toolbox-control.md)

- [Crie um controle de caixa de ferramentas do Windows Forms](../extensibility/creating-a-windows-forms-toolbox-control.md)

## <a name="extend-the-status-bar"></a>Estender a barra de status

Saiba como ler e escrever na barra de status e na barra de progresso, e como fornecer animações e outras uI: [Estender a barra de status](../extensibility/extending-the-status-bar.md).

::: moniker range="vs-2017"

## <a name="create-custom-start-pages"></a>Criar páginas iniciais personalizadas

Aprenda a fazer sua própria página inicial, seja do zero ou a partir de uma amostra de página inicial para download: [Crie uma página inicial personalizada](../extensibility/creating-a-custom-start-page.md).

::: moniker-end

## <a name="write-to-the-activity-log"></a>Escreva para o registro de atividades

Saiba como escrever no Registro de Atividades: [Como: Usar o registro de atividades](../extensibility/how-to-use-the-activity-log.md).
