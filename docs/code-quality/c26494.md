# <a name="c26494-var_use_before_init"></a>C26494 VAR_USE_BEFORE_INIT

A variável '% Variable% ' não foi inicializada. Sempre inicialize um objeto. Ver [Type.5 de diretrizes principais do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
