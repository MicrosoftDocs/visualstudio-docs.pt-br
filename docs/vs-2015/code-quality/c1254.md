# <a name="c1254"></a>C1254
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C1254: Incompatibilidade de versão do plug-in: versão não corresponde à versão do driver do PREfast  
  
 A ferramenta de análise de código relata esse aviso quando há um erro interno com a versão do plug-in, não no código que está sendo analisado.
