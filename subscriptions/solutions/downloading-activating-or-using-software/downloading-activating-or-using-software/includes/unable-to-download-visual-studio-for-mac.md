## <a name="im-unable-to-download-or-activate-visual-studio-for-mac"></a>Não consigo baixar ou ativar Visual Studio para Mac

Como assinante do Visual Studio, você tem acesso ao IDE do Visual Studio. Visual Studio para Mac é um download separado que você pode encontrar na [página de downloads](https://my.visualstudio.com/Downloads) do portal de assinaturas.  

## <a name="find-and-download-visual-studio-for-mac"></a>Localizar e baixar Visual Studio para Mac 
1. Entre no portal de [assinaturas](https://my.visualstudio.com/benefits)e veja uma [exibição filtrada](https://my.visualstudio.com/Downloads?q=Visual%20Studio%20for%20mac&pgroup=) de downloads de Visual Studio para Mac disponíveis. Oferecemos apenas as versões mais recentes do produto. 
2. Siga as instruções de download e de configuração. 
1. Usando o email ao qual sua assinatura está atribuída, você poderá entrar imediatamente no IDE após a instalação, nenhuma chave do produto é necessária.

## <a name="no-download-available"></a>Nenhum download disponível 
Os downloads de software disponíveis dependem da assinatura que você está usando. Se você não vir o download que está procurando, ele poderá não estar disponível para você. Para verificar quais assinaturas estão disponíveis para você, faça logon na [página assinaturas](https://my.visualstudio.com/subscriptions) do Portal. Você pode [baixar uma lista extensa](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)   de títulos de software disponíveis para todos os tipos de assinatura.
