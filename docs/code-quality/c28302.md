# <a name="c28302"></a>C28302
aviso C28302: para C++ o parâmetro de referência < parameter_name >, um operador `_Deref_` extra foi encontrado em > \<annotation.

 Esse aviso é relatado quando um nível extra de `_Deref_` é usado em um parâmetro.

 SAL2 não exige o uso de um nível extra de `_Deref_` ao lidar com parâmetros de referência. Essa anotação específica não é ambígua e é interpretada corretamente, mas deve ser corrigida.

 Frequentemente, isso pode ser corrigido simplesmente removendo a anotação `__deref` mais antiga e usando a sintaxe SAL2. Às vezes, pode ser necessário usar `_At_` para fazer referência ao objeto específico a ser anotado.
