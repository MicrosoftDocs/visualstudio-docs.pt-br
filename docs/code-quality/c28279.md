# <a name="c28279"></a>C28279
Aviso C28279: para símbolo, um 'Início' foi encontrado sem um 'fim' correspondente

 A linguagem de anotação suporta um begin e end (`{` e `}` em C) construção e o emparelhamento ficou desbalanceada. Essa situação pode ser evitada se as macros são usadas.
