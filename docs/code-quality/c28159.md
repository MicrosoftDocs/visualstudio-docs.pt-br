# <a name="c28159"></a>C28159
C28159 de aviso: Considere usar outra função em vez disso.

 Esse aviso é relatado para os drivers que estão sugerindo que você use uma chamada de função preferencial que seja semanticamente equivalente à função que o driver está chamando. Esta é uma mensagem de aviso geral; a anotação `__drv_preferredFunction` foi usada (possivelmente com uma anotação condicional `__drv_when` ()) para sinalizar uma prática de codificação inadequada.

## <a name="example"></a>Exemplo
 O exemplo de código a seguir gera este aviso:

```
char buff[MAX_PATH];

OemToChar(buff, input);

    // if strlen(input) > MAX_PATH
....// leads to buffer overrun
```

 O exemplo de código a seguir evita esse aviso:

```
char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
```
