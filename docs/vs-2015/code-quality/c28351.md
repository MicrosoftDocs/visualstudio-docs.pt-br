# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28351: A anotação \<anotação > descreve onde um valor dinâmico (uma variável) não pode ser usado na condição.  
  
 Esse aviso é relatado quando uma anotação é aplicada em que o tipo de C/C++ está sendo inspecionado.  
  
 A expressão no `_When_` deve ser avaliada como uma constante. O `_When_` será ignorado.
