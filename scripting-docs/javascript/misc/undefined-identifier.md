# <a name="undefined-identifier"></a>Identificador indefinido
Você tentou usar um identificador que o compilador de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] não reconhece. O valor indefinido é retornado sempre que você usa:  
  
- uma variável que não existe,  
  
- uma variável que foi declarada, mas nunca tinha um valor atribuído a ela,  
  
- uma propriedade de objeto que não existe.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declare a variável com uma instrução **var** (como em `var` x;).
