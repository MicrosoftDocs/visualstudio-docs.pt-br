# <a name="c28225"></a>C28225
aviso C28225: não foi possível encontrar o número correto de parâmetros necessários na anotação

 Esse aviso indica que a anotação nomeada é usada com o número incorreto de parâmetros. Isso não deve ser possível se as macros padrão estiverem sendo usadas para anotações; Esse aviso não deve ser visto no uso típico.
