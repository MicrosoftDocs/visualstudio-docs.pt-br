# <a name="visual-studio-dev-essentials"></a>Visual Studio Dev Essentials

O Visual Studio Dev Essentials fornece tudo o que você precisa para compilar e implantar seu aplicativo em qualquer plataforma. Com ferramentas de ponta, o poder da nuvem, treinamento e suporte, é o nosso programa gratuito para desenvolvedores mais abrangente de todos os tempos.

Visite [Dev Essentials](https://visualstudio.microsoft.com/dev-essentials/) para saber mais sobre tudo o que está incluído em uma associação do Visual Studio Dev Essentials.

## <a name="opt-in-to-visual-studio-dev-essentials"></a>Aceitar o Visual Studio Dev Essentials

Quando estiver pronto para se registrar, basta clicar no botão [Registrar ou acessar agora](https://my.visualstudio.com/Benefits?wt.mc_id=o~msft~vscom~devessentials-hero~mt689&campaign=o~msft~vscom~devessentials-hero~mt689).

E isso é tudo.

## <a name="opt-out-of-visual-studio-dev-essentials"></a>Recusar o Visual Studio Dev Essentials

Se você quiser recusar o programa Visual Studio Dev Essentials, execute estas etapas:

1. Acesse [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs)
2. Entre em sua assinatura.  (Para obter ajuda, [entre](signing-in.md)).
3. Após a conexão bem-sucedida, será exibida uma lista de assinaturas atuais.
4. Na linha associada ao Visual Studio Dev Essentials, clique em **Sair do programa** na coluna Links na extrema direita.
5. Uma janela pop-up será exibida. Selecione **Confirmar**.

    > [!NOTE]
    > Voltar ao programa Visual Studio Dev Essentials é fácil.  Basta seguir as etapas 1 a 5 e clicar no botão **Ingressar no Visual Studio Dev Essentials** na parte superior da coluna Links. Depois, clique em **Confirmar**.
