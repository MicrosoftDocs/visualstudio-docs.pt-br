# <a name="c28304"></a>C28304
C28304 de aviso: Para C++ o parâmetro de referência < parameter_name >, um operador `_Notref_` colocado incorretamente foi encontrado para > \<token.

 O operador `_Notref_` só deve ser usado em circunstâncias especiais envolvendo C++ parâmetros de referência e somente em macros fornecidas pelo sistema. Ele deve ser seguido imediatamente por um operador `_Deref_` ou uma anotação funcional.
