# <a name="xml-documentation-comments-javascript"></a>Comentários de documentação XML (JavaScript)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

No JavaScript, você pode criar documentação para seu código, incluindo marcas XML em campos de comentário especiais no código-fonte. Essas marcas XML são usadas pelo IntelliSense para fornecer informações sobre objetos JavaScript.

 O local dos comentários do código varia dependendo da marca específica.

 Há suporte para as seguintes marcas:

- [\<deprecated>](../ide/deprecated-javascript.md)

- [\<field>](../ide/field-javascript.md)

- [\<loc>](../ide/loc-javascript.md)

- [\<param>](../ide/param-javascript.md)

- [\<returns>](../ide/returns-javascript.md)

- [\<signature>](../ide/signature-javascript.md)

- [\<summary>](../ide/summary-javascript.md)

- [\<value>](../ide/value-javascript.md)

- [\<var>](../ide/var-javascript.md)
