# <a name="c26138"></a>C26138
warning C26138: A suspensão de uma co-rotina, mantendo o bloqueio \<bloqueio >.

Aviso C26138 avisa quando uma co-rotina é suspensa enquanto mantém um bloqueio. Em geral, não sabemos quanto tempo será uma corrotina permanecem no estado suspenso, portanto, esse padrão pode resultar em mais seções críticas do que o esperado.

## <a name="example"></a>Exemplo
 O código a seguir gerará C26138.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}

generator<int> mutex_acquiring_generator_report_once() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}
```

## <a name="example"></a>Exemplo
 O código a seguir corrigirá esses avisos.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator2() {
  {
    global_m.lock();
    ++var;
    global_m.unlock();
  }
  co_yield 1;                           // no 26138, global_m is already released above.
}
```
