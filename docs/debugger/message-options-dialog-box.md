# <a name="message-options-dialog-box"></a>Caixa de diálogo Opções da Mensagem
Use essa caixa de diálogo para selecionar quais mensagens são listadas na [exibição de mensagens](../debugger/messages-view.md). Para exibir essa caixa de diálogo, escolha **mensagens de Log** da **Spy** menu.

## <a name="in-this-section"></a>Nesta seção
 [Guia do Windows, a caixa de diálogo de opções de mensagem](../debugger/windows-tab-message-options-dialog-box.md) usado para selecionar os tipos de mensagens para listar. Inclui a ferramenta de localização.

 [Guia, caixa de diálogo de opções de mensagem mensagens](../debugger/messages-tab-message-options-dialog-box.md) usado para selecionar os tipos de mensagem para a exibição.

 [Guia saída, caixa de diálogo de opções de mensagem](../debugger/output-tab-message-options-dialog-box.md) usado para especificar quais dados de mensagem para exibir.
