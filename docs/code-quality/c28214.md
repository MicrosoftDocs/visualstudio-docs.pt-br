# <a name="c28214"></a>C28214
warning C28214: Os nomes do parâmetro de atributo devem ser p1...p9

 Este aviso indica que, quando você constrói uma declaração de anotação, os nomes de parâmetro são limitados para p1... p9. Esse aviso nunca deve ocorrer em uso normal.
