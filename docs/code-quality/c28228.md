# <a name="c28228"></a>C28228
Aviso C28228: parâmetro de anotação: não é possível usar tipo nas anotações

 Esse aviso indica que um parâmetro é do tipo que não tem suporte. Anotações só podem usar um conjunto limitado de tipos como parâmetros. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
