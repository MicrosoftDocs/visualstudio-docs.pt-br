# <a name="code-magazine-included-in-visual-studio-subscriptions"></a>Revista CODE incluída nas assinaturas do Visual Studio

Code Magazine é uma revista líder em desenvolvedores de software independenteque cobre tecnologias de desenvolvimento em todas as plataformas.  As assinaturas selecionadas do Visual Studio são elegíveis para receber uma assinatura de um ano para as edições impressas ou digitais da revista.

## <a name="activation-steps"></a>Etapas de ativação
O processo para configurar sua assinatura do CODE Magazine é simples.  Basta seguir estas etapas:

1. Visitar[https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
2. Localize o azulejo da Revista CODE na seção Desenvolvimento Ou Educação Profissional do seu portal e clique **em Obter Código**.
   > [!div class="mx-imgBorder"]
   > ![Telha de revista CODE](_img/vs-code-magazine/vs-code-magazine-tile.png "Azulejo de revista CODE")
3. Você verá uma mensagem de que seu código foi recuperado.  Clique em **Ativar** para continuar.  (Se você não está pronto para ativar, tudo bem.  Você sempre pode voltar para a telha para ativar.  O link "Obter código" lerá "Ativar" depois de recuperar seu código.)
   > [!div class="mx-imgBorder"]
   > ![Código da revista CODE recuperado](_img/vs-code-magazine/vs-code-magazine-success.png "Código recuperado com sucesso")
4. Você será redirecionado para a página de aterrissagem da REVISTA CODE. Selecione seu país e escolha como você gostaria de receber a revista. 
   > [!NOTE]
   > Os assinantes do Visual Studio Dev Essentials têm acesso apenas à assinatura digital. 
5. Com base em suas seleções, você verá uma nota mostrando o preço da sua assinatura junto com o valor do cupom que foi aplicado.
   > [!NOTE]
   > Para assinaturas impressas:
   > - Assinantes fora dos Estados Unidos podem incorrer em taxas de transporte. 
   > - Por favor, permita várias semanas para a entrega de sua primeira edição impressa.
      > [!div class="mx-imgBorder"]
      > ![Detalhes da assinatura da revista CODE](_img/vs-code-magazine/vs-code-magazine-details.png "Detalhes e preço da assinatura")
6. Em seguida, você precisará fornecer informações ao seu assinante e um endereço de entrega.  (O endereço de entrega é necessário até mesmo para assinaturas digitais, pois suporta processos de preenchimento, seleção de ofertas e acesso digital.)
7. Se você mora fora dos Estados Unidos e optou por receber uma assinatura de revista impressa, você pode ser solicitado a fornecer um endereço de cobrança e informações de pagamento para cobrir os custos do envio. 
8. Você pode optar pelo CODE HTML Newsletter clicando na caixa de seleção.
   > [!div class="mx-imgBorder"]
   > ![Informações sobre assinantes da Revista CODE](_img/vs-code-magazine/vs-code-magazine-subscriber-info.png "Informações do assinante e entregar endereço")
9. Clique **em Colocar ordem** para completar sua assinatura.  
Depois de completar sua ordem de assinatura, você verá uma página de confirmação, incluindo https://codemag.com/my/fulfillmentum link que você pode usar para ver quais problemas você recebeu: .

## <a name="frequently-asked-questions"></a>Perguntas frequentes
### <a name="q-if-the-subscription-is-free-why-am-i-being-asked-for-a-credit-card"></a>P: Se a assinatura é gratuita, por que me pediram um cartão de crédito?  
R: É proibitivo enviar cópias impressas gratuitas internacionalmente.  Os assinantes que escolherem a edição impressa da revista CODE e residirem fora dos Estados Unidos serão cobrados uma taxa pelo transporte. Para evitar cobranças de frete, escolha a versão digital da revista. 

### <a name="q-why-do-i-need-to-provide-a-delivery-address-for-a-digital-subscription"></a>P: Por que eu preciso fornecer um endereço de entrega para uma assinatura digital?
R: A REVISTA CODE coleta seu endereço, independentemente de a assinatura ser para uma assinatura impressa ou digital para apoiar o cumprimento, a seleção de ofertas e os processos de acesso digital.  Para obter mais informações sobre o que a CODE Magazine faz com seus dados, consulte sua [Política de Privacidade](https://www.codemag.com/Home/Privacy).
