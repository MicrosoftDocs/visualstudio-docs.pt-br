# <a name="getting-started-with-source-control-vspackages"></a>Introdução ao VSPackages de controle do código-fonte
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Um VSPackage de controle de origem é uma alternativa mais avançada para o plug-in de controle do código-fonte. (Consulte [criando um plug-in de controle do código-fonte](../../extensibility/internals/creating-a-source-control-plug-in.md) para obter mais informações sobre plug-ins de controle do código-fonte.) Um VSPackage de controle de origem permite o controle total sobre o modelo de controle do código-fonte, os recursos e a interface do usuário e se integra ao [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] ambiente como um VSPackage.
