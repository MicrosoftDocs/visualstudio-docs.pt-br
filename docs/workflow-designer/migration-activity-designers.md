# <a name="migration-activity-designers"></a>Designers de atividade de migração

Os designers de atividade de migração são usados para chamar atividades do fluxo de trabalho 3.0/3.5 de um fluxo de trabalho que tem como destino .NET Framework 4 ou posterior.

Os tópicos contidos nesta seção fornecem orientação sobre como usar os designers de atividade de Designer de Fluxo de Trabalho.
