# <a name="c28208"></a>C28208
Aviso C28208: função \<função > foi definido anteriormente com uma lista de parâmetros diferentes no \<arquivo > (\<linha >). Algumas ferramentas de análise produzirão resultados incorretos

 Esse aviso é relatado quando a definição de uma função conhecidos não corresponde a outra ocorrência.
