# <a name="extend-user-settings-and-options"></a>Estender as configurações e opções do usuário
No Visual Studio existem duas maneiras principais de manter as preferências do usuário. As páginas**Opções de** **ferramentas** > permitem que os usuários definam seus próprios valores para muitos tipos diferentes de configurações, tanto gerais quanto específicas do idioma. As configurações do usuário permitem que os usuários especifiquem as propriedades gerais de configuração do Visual Studio. Para obter mais informações sobre páginas de opções, consulte [Opções e páginas de opções](../extensibility/internals/options-and-options-pages.md). Para obter mais informações sobre as configurações do usuário, consulte [Suporte para configurações do usuário](../extensibility/internals/support-for-user-settings.md).

- [Crie uma página de opções](../extensibility/creating-an-options-page.md) Explica como criar uma página de opções.

- [Criar uma categoria de configurações](../extensibility/creating-a-settings-category.md) Explica como criar uma categoria de configurações.

- [Use a loja de configurações](../extensibility/using-the-settings-store.md) Explica como usar a loja de configurações.

- [Obtenha informações de serviço na loja de configurações](../extensibility/getting-service-information-from-the-settings-store.md) Explica como obter os serviços disponíveis na loja de configurações.

- [Escreva para a loja de configurações do usuário](../extensibility/writing-to-the-user-settings-store.md) Explica como escrever na loja de configurações do usuário.
