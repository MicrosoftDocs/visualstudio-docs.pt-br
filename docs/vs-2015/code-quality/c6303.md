# <a name="c6303"></a>C6303
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C6303: formatação de cadeia de caracteres incompatível: cadeia de caracteres largo passada como parâmetro \<number> quando a cadeia de caracteres é necessária na chamada para \<function>  
  
 Esse aviso indica que a cadeia de caracteres de formato especifica que uma cadeia de caracteres é necessária. No entanto, uma cadeia de caracteres larga está sendo passada. Esse defeito provavelmente causará uma falha ou uma corrupção de algum tipo.  
  
## <a name="example"></a>Exemplo  
 O código de exemplo a seguir gera este aviso:  
  
```  
#include <stdio.h>  
  
void f()  
{  
  wchar_t buff[5] = L"hi";  
  
  printf("%s", buff);  
}  
```  
  
 Para corrigir esse aviso, use `%ls` conforme mostrado no código de exemplo a seguir:  
  
```  
#include <stdio.h>  
  
void f()  
{  
  wchar_t buff[5] = L"hi";  
  
  printf("%ls", buff);  
}  
```  
  
 O código de exemplo a seguir usa a função `printf_s` de manipulação de cadeia de caracteres segura para corrigir este aviso:  
  
```  
#include <stdio.h>  
  
void f()  
{  
  wchar_t buff[5] = L"hi";  
  
  printf_s("%ls",buff);  
}  
```
