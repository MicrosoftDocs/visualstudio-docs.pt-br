# <a name="c1256"></a>C1256
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C1256: PCH deve ser gravado e lido.  
  
 A ferramenta de análise de código relata esse aviso quando há um erro interno na ferramenta, não no código que está sendo analisado.
