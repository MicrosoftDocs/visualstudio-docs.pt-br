# <a name="supplying-xml-code-comments"></a>Fornecendo comentários de código XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

É possível disponibilizar os comentários de arquivos de código-fonte como o IntelliSense. Ao adicionar a opção /doc à linha de comando do compilador do Visual Basic, C# ou C++, é gerado um arquivo .xml que serve como base para a documentação do IntelliSense.

 Para obter mais informações sobre como documentar o código no Visual Basic, consulte [Como: criar documentação XML](https://msdn.microsoft.com/library/27b5b06c-09b9-496a-8245-f9542d846230).

 Para obter mais informações sobre como documentar o código no C#, consulte [Comentários da documentação XML](https://msdn.microsoft.com/library/803b7f7b-7428-4725-b5db-9a6cff273199).

 Para obter mais informações sobre como documentar o código no C++, consulte [Comentários](https://msdn.microsoft.com/library/6fcb906c-c264-4083-84bc-373800b2e514) e [Documentação XML](https://msdn.microsoft.com/library/a1aec1c5-b2d1-4c74-83ae-1dbbbb76b506).
