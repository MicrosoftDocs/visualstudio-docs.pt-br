# <a name="message-options-dialog-box"></a>Caixa de diálogo Opções da Mensagem
Use essa caixa de diálogo para selecionar quais mensagens estão listadas no [modo de exibição de mensagens](../debugger/messages-view.md). Para exibir essa caixa de diálogo, escolha **mensagens de log** no menu do **Spy** .

## <a name="in-this-section"></a>Nesta seção
 [Guia Windows, caixa de diálogo opções de mensagem](../debugger/windows-tab-message-options-dialog-box.md) Usado para selecionar os tipos de mensagens a serem listadas. Inclui a ferramenta Finder.

 [Guia mensagens, caixa de diálogo opções de mensagem](../debugger/messages-tab-message-options-dialog-box.md) Usado para selecionar tipos de mensagem para exibição.

 [Guia saída, caixa de diálogo opções de mensagem](../debugger/output-tab-message-options-dialog-box.md) Usado para especificar quais dados de mensagem exibir.
