# <a name="c26462-useconstpointerforvariable"></a>C26462 USE_CONST_POINTER_FOR_VARIABLE
  O valor apontado por '% variable %' é atribuído somente uma vez, marcá-la como um ponteiro para `const`. Ver [con.4 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
