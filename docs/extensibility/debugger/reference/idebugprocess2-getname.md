# <a name="idebugprocess2getname"></a>IDebugProcess2::GetName
Obtém o título, nome amigável ou nome do arquivo do processo.

## <a name="syntax"></a>Sintaxe

```cpp
HRESULT GetName( 
   GETNAME_TYPE  gnType,
   BSTR*         pbstrName
);
```

```csharp
int GetName( 
   enum_GETNAME_TYPE  gnType,
   out string         pbstrName
);
```

## <a name="parameters"></a>parâmetros
`gnType`\
[em] Um valor da enumeração [GETNAME_TYPE](../../../extensibility/debugger/reference/getname-type.md) que especifica qual tipo de nome deve retornar.

`pbstrName`\
[fora] Retorna o nome do processo.

## <a name="return-value"></a>Valor retornado
 Se for `S_OK`bem sucedido, retorna; caso contrário, retorna um código de erro.
