# <a name="lib-task"></a>Tarefa LIB
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Encapsula a ferramenta Gerenciador de Biblioteca de 32 bits da Microsoft, lib.exe. O Gerenciador de Biblioteca cria e gerencia uma biblioteca de arquivos-objetos de formato COFF. O Gerenciador de Biblioteca também pode criar arquivos de exportação e importar bibliotecas para referenciar definições exportadas. Para obter mais informações, consulte [lib Reference](https://msdn.microsoft.com/library/ecc7f643-bbd4-47a3-8dc6-b360f880db91) e [running lib](https://msdn.microsoft.com/library/d54f5c81-7147-4b2c-a8db-68ce6eb1eabd).
