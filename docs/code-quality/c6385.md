# <a name="c6385"></a>C6385
aviso C6385: dados inválidos: acessando o nome de \<buffer >, o tamanho legível é \<size1 > bytes, mas \<size2 > bytes podem ser lidos: linhas: x, y

 Esse aviso indica que a extensão legível do buffer especificado pode ser menor do que o índice usado para ler a partir dele. As tentativas de ler dados fora do intervalo válido levam à saturação do buffer.

## <a name="example"></a>Exemplo
 O código a seguir gera este aviso:

```
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

 Para corrigir esse aviso, use o seguinte código:

```
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
