# <a name="c26460-useconstreferencearguments"></a>C26460 USE_CONST_REFERENCE_ARGUMENTS

  O argumento de referência '% % do argumento' para a função '% % de função' pode ser marcado como `const`. Ver [con.3 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
