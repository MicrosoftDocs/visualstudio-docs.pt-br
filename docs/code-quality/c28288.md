# <a name="c28288"></a>C28288
C28288 de aviso: Para função, Erro de sintaxe na anotação \_At\_() (nome de parâmetro inválido)

 A ferramenta de análise de código relata esse aviso quando a anotação `SAL_at` (`__drv_at`) é usada e a expressão de parâmetro não pode ser interpretada no contexto atual. Isso pode incluir o uso de um parâmetro ou nome de membro digitado incorretamente ou uma grafia incorreta das palavras-chave "Return" ou "This".
