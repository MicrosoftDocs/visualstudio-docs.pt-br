# <a name="c26465-no_const_cast_unnecessary"></a>C26465 NO_CONST_CAST_UNNECESSARY

Não use `const_cast` para converter `const`. `const_cast` não é necessário; constness ou volatility não está sendo removido por essa conversão. Ver [Type.3 de diretrizes principais do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-constcast).
