# <a name="c6515"></a>C6515
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso c6515: anotação inválida: \<nome > propriedade pode ser usada somente em valores do tipo de ponteiro  
  
 Este aviso indica que uma propriedade para uso em ponteiros foi aplicada a um tipo de não ponteiro. Para obter uma lista de propriedades de anotação, consulte [propriedades de anotação](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Exemplo  
 O código a seguir gera este aviso:  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 Para corrigir este aviso, use o seguinte código:  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
