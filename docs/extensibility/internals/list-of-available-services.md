# <a name="list-of-available-services"></a>Lista de serviços disponíveis

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] e o SDK do Visual Studio dá suporte aos serviços a seguir. Alguns pacotes oferecem seus próprios serviços que não estão listados aqui — por exemplo, os serviços de linguagem não têm um único GUID de serviço. Você deve usar o nome do idioma para localizar o GUID do serviço de idioma no registro.

Use os GUIDs de serviço listados aqui ou obtidos de alguma outra fonte (por exemplo, serviços de linguagem) para obter a interface primária ou as interfaces mostradas com cada serviço.
