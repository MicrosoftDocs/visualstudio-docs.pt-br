# <a name="javascript-language-reference"></a>{1&gt;{2&gt;Referência da linguagem JavaScript&lt;2}&lt;1}

O JavaScript é uma linguagem de script que pode ser inserida em páginas da Web e em outros aplicativos.  
  
A implementação do JavaScript da Microsoft é compatível com a edição da linguagem ECMAScript 5º. Ela também fornece recursos adicionais que não fazem parte dos padrões ECMA.

> [!IMPORTANT]
> A Microsoft ingressou no esforço de toda a Comunidade para tornar o [MDN Web docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) a referência de desenvolvimento One-Stop da Web, começando pelo redirecionamento de todas as mais de 500 páginas da referência de API JavaScript da Microsoft do [docs.Microsoft.com](https://docs.microsoft.com) para seus MDN e portando nossa referência de extensões do js herdada para fins de serviço e histórico.
