# <a name="aspnet-core-web-app-deployment"></a>Implantação de aplicativo Web do ASP.NET Core

Uma etapa importante no desenvolvimento de aplicativos Web é a publicação (implantação). Você pode publicar em um ambiente remoto, para que outras pessoas possam usar o aplicativo, ou para fins de teste. O Visual Studio para Mac oferece duas opções para publicar seu aplicativo. A opção Publicar no Serviço de Aplicativo do Azure permite publicar o aplicativo Web no Serviço de Aplicativo do Azure. A opção Publicar na Pasta permite publicar o aplicativo em uma pasta.

## <a name="publish-to-azure-app-service"></a>Publicar no Serviço de Aplicativo do Azure

Usando o Visual Studio para Mac, você pode publicar o aplicativo Web diretamente no Serviço de Aplicativo do Azure. Para publicar o aplicativo Web do ASP.NET Core no Serviço de Aplicativo do Azure, confira o artigo [Publicar no Serviço de Aplicativo do Azure](publish-app-svc.md).

## <a name="publish-to-folder"></a>Publicar na Pasta

Você também pode publicar seus aplicativos Web do ASP.NET Core em uma pasta. Para saber mais sobre a publicação do aplicativo Web em uma pasta, confira [Publicar na Pasta](publish-folder.md).
