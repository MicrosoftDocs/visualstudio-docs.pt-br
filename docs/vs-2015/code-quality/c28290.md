# <a name="c28290"></a>C28290
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28290: a anotação para a função contém mais externos do que o número real de parâmetros  
  
 A ferramenta de análise de código relata esse aviso quando a anotação da função contém mais externos do que o número real de parâmetros.
