# <a name="web-projects"></a>Projetos Web
Um sistema de projetos Web é um sistema de projetos que cria projetos web. Projetos web, por sua vez, criam aplicativos web. Você pode programar uma página da Web [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] usando [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]linguagens de programação tradicionais, como ou , para coletar e processar informações de um usuário, armazená-la em um banco de dados, e assim por diante. Páginas da Web com código associado são chamadas páginas da Web inteligentes.
