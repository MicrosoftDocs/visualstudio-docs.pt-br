# <a name="idebugdocumenttextauthor-interface"></a>Interface IDebugDocumentTextAuthor
Fornecido por documentos de texto que suporte a edição.  
  
 Além dos métodos herdados de `IDebugDocumentText`, o `IDebugDocumentTextAuthor` interface expõe os métodos a seguir.  
  
## <a name="methods-in-vtable-order"></a>Métodos na ordem de Vtable  
  
|Método|Descrição|  
|------------|-----------------|  
|[IDebugDocumentTextAuthor::InsertText](../../winscript/reference/idebugdocumenttextauthor-inserttext.md)|Insere um novo texto no documento.|  
|[IDebugDocumentTextAuthor::RemoveText](../../winscript/reference/idebugdocumenttextauthor-removetext.md)|Remove o texto do documento.|  
|[IDebugDocumentTextAuthor::ReplaceText](../../winscript/reference/idebugdocumenttextauthor-replacetext.md)|Substitui o texto no documento.|
