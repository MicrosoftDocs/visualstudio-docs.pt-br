# <a name="invalid-number"></a>Número inválido
Número inválido (ErrbadNumber 1013)  
  
 Você tentou usar um número que não é válido em JavaScript.  
  
 Para obter informações sobre números válidos, consulte o [objeto Number](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number).
