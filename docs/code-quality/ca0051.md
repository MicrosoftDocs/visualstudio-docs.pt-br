# <a name="ca0051"></a>CA0051

Nenhuma regra foi selecionada.

Esse erro indica que a ferramenta de análise de código gerenciado está habilitada, mas nenhuma regra foi selecionada. Para corrigir esse erro, selecione regras apropriadas.
