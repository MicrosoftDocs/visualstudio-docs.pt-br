# <a name="c28163"></a>C28163
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28163: a função nunca deve ser chamada de dentro de um bloco try/Except  
  
 Esse aviso é relatado quando uma função é de um tipo que nunca deve ser colocado em um `try/except`  bloco em um `try/except` bloco. A ferramenta de análise de código encontrou pelo menos um caminho no qual a função chamada estava dentro de um `try/except` bloco.
