# <a name="ca0501"></a>CA0501

Não é possível ler o relatório de saída.

Isso ocorre quando o relatório de saída não pode ser gravado devido a um dos seguintes motivos:

-   Disco cheio

-   Nenhuma permissão para gravar no diretório

-   O diretório não existe
