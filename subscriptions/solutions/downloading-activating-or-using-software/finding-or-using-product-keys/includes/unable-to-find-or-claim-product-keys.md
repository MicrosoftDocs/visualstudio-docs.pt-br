## <a name="im-unable-to-find-or-claim-a-product-key"></a>Não consigo encontrar ou reivindicar uma chave do produto (Product Key).

As chaves de produto disponíveis para você dependem da assinatura do Visual Studio que você está usando.  

## <a name="view-a-list-of-available-product-keys"></a>Exibir uma lista de chaves de produto disponíveis 

A página [chaves do produto](https://my.visualstudio.com/productkeys)   lista todas as chaves disponíveis para você na agregação. Você pode editar a coluna observações para identificar quando e por que a chave foi solicitada. Há diferentes tipos de chave, dependendo do produto. Algumas chaves que você precisará reivindicar e outras ficarão visíveis para você imediatamente. 

Se você não vir uma chave ou produto, é possível que o produto não esteja disponível para seu nível de assinatura. Você pode [baixar uma lista extensa](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx) de títulos de software disponíveis para todos os tipos de assinatura. Para verificar quais tipos estão atribuídos a você, exiba suas assinaturas no [portal de assinaturas](https://my.visualstudio.com/subscriptions).  

## <a name="more-information-on-product-key-types"></a>Mais informações sobre tipos de chave do produto

Leia mais sobre os [tipos de chave de produto e reivindicando chaves de produto](https://docs.microsoft.com/visualstudio/subscriptions/find-keys).
