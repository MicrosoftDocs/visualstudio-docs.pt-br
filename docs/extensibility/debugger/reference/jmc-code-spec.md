# <a name="jmc_code_spec"></a>JMC_CODE_SPEC
Esta estrutura é usada para definir as informações do JustMyCode para um módulo.

## <a name="syntax"></a>Sintaxe

```cpp
typedef struct _JMC_CODE_SPEC {
    BOOL fIsUserCode;
    BSTR bstrModuleName;
} JMC_CODE_SPEC;
```

```csharp
public struct JMC_CODE_SPEC {
    public int    fIsUserCode;
    public string bstrModuleName;
};
```

## <a name="members"></a>Membros
`fIsUserCode`\
Não-zero`TRUE`( ) se o módulo for considerado código de usuário; caso contrário,`FALSE`zero ( ) se o módulo deve ser tratado como código externo e não para ser depurado.

`bstrModuleName`\
Nome do módulo em questão.

## <a name="remarks"></a>Comentários
Essa estrutura é passada como uma lista dessas estruturas para o método [SetJustMyCodeState.](../../../extensibility/debugger/reference/idebugengine3-setjustmycodestate.md)
