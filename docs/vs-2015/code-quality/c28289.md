# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28289: Para a função: ReadableTo ou WritableTo não tinha uma especificação de limite como um parâmetro  
  
 A ferramenta de análise de código relata esse aviso quando a anotação de parâmetro de função/é miscoded conforme observado.
