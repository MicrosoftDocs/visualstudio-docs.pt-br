# <a name="add-commands-to-toolbars"></a>Adicionar comandos às barras de ferramentas
Uma barra de ferramentas é uma tira horizontal ou vertical que contém botões ligados a comandos. Você pode adicionar seus próprios comandos às barras de ferramentas em muitas janelas diferentes no Visual Studio: a janela principal do Visual Studio, o Solution Explorer e suas próprias janelas de ferramentas personalizadas.

 Esta seção mostra uma série de coisas que você pode fazer com comandos em barras de ferramentas.

## <a name="in-this-section"></a>Nesta seção
- [Adicione uma barra de ferramentas](../extensibility/adding-a-toolbar.md) Explica como adicionar uma barra de ferramentas ao conjunto de barras de ferramentas na janela principal do Visual Studio.

- [Adicione uma barra de ferramentas a uma janela de ferramenta](../extensibility/adding-a-toolbar-to-a-tool-window.md) Explica como adicionar uma barra de ferramentas a uma janela de ferramenta.

- [Adicione um comando à barra de ferramentas do Solution Explorer](../extensibility/adding-a-command-to-the-solution-explorer-toolbar.md) Explica como adicionar um comando à barra de ferramentas no Solution Explorer.

- [Adicione um controlador de menu a uma barra de ferramentas](../extensibility/adding-a-menu-controller-to-a-toolbar.md) Explica como adicionar um controlador de menu a uma barra de ferramentas.

- [Adicionar dinamicamente itens de menu](../extensibility/dynamically-adding-menu-items.md) Explica como adicionar itens de menu dinamicamente em diferentes contextos.
