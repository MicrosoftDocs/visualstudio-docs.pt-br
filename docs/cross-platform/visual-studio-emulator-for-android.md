# <a name="visual-studio-emulator-for-android"></a>Emulador do Visual Studio para Android

O Visual Studio Emulador para Android é um aplicativo da área de trabalho para emular um dispositivo Android. Ele foi preterido no Visual Studio 2015 e não tem suporte no Visual Studio 2017 ou Visual Studio 2019. Em vez disso, recomendamos que você use os emuladores do Google Android. Para ver as Visual Studio emulador para Android do Visual Studio 2015, escolha Visual Studio 2015 no seletor de versão da documentação.

> [!IMPORTANT]
> Na maioria dos cenários, o emulador do Google Android é recomendado para uso, em vez do Emulador do Visual Studio para Android:
> - Visual Studio emulador para Android não tem suporte após Visual Studio 2015.
> - As imagens do emulador posteriores à versão 6.0 do Android não estão disponíveis para Visual Studio Emulador para Android.
> - O Google Android Emulator agora é compatível com [Hyper-V](/xamarin/android/get-started/installation/android-emulator/hardware-acceleration#accelerating-with-hyper-v).
> - O C++ para Desenvolvimento de Dispositivo Móvel e Ferramentas do Visual Studio para Apache Cordova funcionam com o Google Android Emulator. Para saber mais, confira [Executar seu aplicativo Apache Cordova no Android](/previous-versions/visualstudio/cross-platform/tools-for-cordova/run-your-app/run-app-android#google-android-emulator). (Você não precisa mais desabilitar o Hyper-V, conforme explicado neste artigo).
>
> Para obter mais informações sobre como configurar e usar o Google Android Emulator, [consulte Android Emulator configuração do](/xamarin/android/get-started/installation/android-emulator/).
