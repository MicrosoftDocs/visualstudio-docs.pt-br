# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28280: para símbolo, um 'fim' foi encontrado sem encontrar uma correspondência 'begin'  
  
 A linguagem de anotação dá suporte a um início e fim (`{` e `}` em C) constructo e o emparelhamento ficou desbalanceada. Essa situação pode ser evitada se as macros são usadas.
