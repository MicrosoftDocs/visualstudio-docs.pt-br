# <a name="c28020"></a>C28020
C28020 de aviso: A expressão \<expr > não é verdadeira nesta chamada

 Esse aviso é relatado quando a expressão \_Satisfies @ no__t-1 listada não é verdadeira. Frequentemente, isso indica um parâmetro incorreto.

 Se isso ocorrer em uma declaração de função, as anotações indicarão uma condição impossível.
