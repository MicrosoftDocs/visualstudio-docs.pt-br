# <a name="c28211"></a>C28211
C28211 de aviso: Nome esperado do contexto estático para SAL_context

 Esse aviso indica que o operando para a anotação `_Static_context_` deve ser o nome de um contexto definido por uma ferramenta. Esse aviso nunca deve ocorrer em uso normal.
