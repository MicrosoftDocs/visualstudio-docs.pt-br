# <a name="iremotedebugapplicationthreadenumstackframes"></a>IRemoteDebugApplicationThread::EnumStackFrames
Retorna um enumerador para os quadros de pilhas associados a esse thread.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT EnumStackFrames(  
   IEnumDebugStackFrames**  ppedsf  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `ppedsf`  
 fora Um enumerador para os quadros de pilha associados a este thread.  
  
## <a name="return-value"></a>Valor retornado  
 O método retorna um `HRESULT`. Os possíveis valores incluem, mas sem limitação, aqueles na tabela a seguir.  
  
|Valor|Descrição|  
|-----------|-----------------|  
|`S_OK`|O método foi bem-sucedido.|  
  
## <a name="remarks"></a>Comentários  
 Esse método deve ser chamado de dentro de um ponto de interrupção. O enumerador de quadros de pilha deve retornar os quadros que começam na parte superior da pilha, começando com o quadro enviado por push mais recentemente.
