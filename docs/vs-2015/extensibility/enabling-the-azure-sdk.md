# <a name="enabling-the-azure-sdk"></a>Habilitar o SDK do Azure
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

O SDK do Azure para .NET ajuda a criar, configurar, compilar, depurar, executar e empacotar aplicativos Web e serviços escalonáveis no Azure.  
  
 Você pode baixar o SDK do Azure na caixa de diálogo **novo projeto** .  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Para habilitar o SDK do Azure  
  
1. No menu **arquivo** , crie um novo projeto. (Clique em **novo** e em **projeto**.)  
  
2. Expanda o nó para **Visual Basic** ou **C#** e clique em **nuvem**.  
  
3. No painel central, selecione **obter SDK do Azure para .net**e clique em **OK**.  
  
     Siga as instruções no instalador.  
  
    > [!NOTE]
    > Para instalar as ferramentas do Azure, você deve ter Serviços de Informações da Internet 7,0 e os componentes de desenvolvimento de aplicativos ASP.NET instalados. Se esses componentes não estiverem instalados, a instalação das ferramentas do Azure solicitará que você os instale.
