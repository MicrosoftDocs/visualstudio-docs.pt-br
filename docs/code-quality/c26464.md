# <a name="c26464-useconstpointerforelements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  Os valores apontados por elementos de matriz '% % de matriz' são atribuídos somente uma vez, marque-os como ponteiro para `const`. Ver [con.4 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
