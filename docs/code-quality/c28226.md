# <a name="c28226"></a>C28226
C28226 de aviso: Anotação não pode ser também um PrimOp (na declaração atual)

 Esse aviso indica que a anotação nomeada está sendo declarada como um PrimOp e também foi declarada anteriormente como uma anotação normal. Isso não deve ser possível se as macros padrão estiverem sendo usadas para anotações; Esse aviso não deve ser visto no uso normal.
