# <a name="iapplicationdebuggeronclose"></a>IApplicationDebugger::onClose
Manipula um evento de fechamento de aplicativo de depuração.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT onClose();  
```  
  
#### <a name="parameters"></a>Parâmetros  
 Esse método não usa nenhum parâmetro.  
  
## <a name="return-value"></a>Valor de retorno  
 O método retorna um `HRESULT`. Os possíveis valores incluem, mas sem limitação, aqueles na tabela a seguir.  
  
|Valor|Descrição|  
|-----------|-----------------|  
|`S_OK`|O método foi bem-sucedido.|  
  
## <a name="remarks"></a>Comentários  
 Este método é chamado quando `IDebugApplication::Close` é chamado.
