# <a name="color-theme-tools"></a>Ferramentas de tema de cores
O SDK do Visual Studio fornece várias ferramentas para ajudá-lo a criar e gerenciar temas de cores.

- A ferramenta [Editor de cores do VSIX](../../extensibility/internals/vsix-color-editor.md) pode criar e Editar cores personalizadas para o Visual Studio.

- A ferramenta do [compilador de cores do VSIX](../../extensibility/internals/vsix-color-compiler.md) converte arquivos *. xml* de tema do Visual Studio existentes em arquivos *. pkgdef* para sua extensão.
