# <a name="idebugasyncoperationcallback-interface"></a>Interface IDebugAsyncOperationCallBack
Fornece eventos de status relacionados ao andamento da avaliação de uma interface `IDebugAsyncOperation`.  
  
## <a name="methods"></a>Métodos  
 Além dos métodos herdados de `IUnknown`, o `IDebugAsyncOperationCallBack` interface expõe os métodos a seguir.  
  
|Método|Descrição|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Sinaliza que um resultado está disponível em uma operação assíncrona de depuração.|
