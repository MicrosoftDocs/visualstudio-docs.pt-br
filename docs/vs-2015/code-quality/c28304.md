# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28304: para o parâmetro de referência do C++ <parameter_name>, um operador incorretamente posicionado `_Notref_` foi encontrado para \<token> .  
  
 O `_Notref_` operador só deve ser usado em circunstâncias especiais envolvendo parâmetros de referência do C++ e somente em macros fornecidas pelo sistema. Ele deve ser seguido imediatamente por um `_Deref_` operador ou uma anotação funcional.
