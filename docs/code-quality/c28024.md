# <a name="c28024"></a>C28024
aviso C28024: o ponteiro de função que está sendo atribuído é anotado com a classe de função \<class >, que não está contida nas classes de função \<classlist >.

 Esse aviso é relatado quando ambas as funções foram anotadas com uma classe de função, mas as classes não correspondem.

 Esse aviso é emitido quando um ponteiro de função tem uma anotação `_Function_class_` que especifica que somente as funções de uma determinada classe funcional devem ser atribuídas a ela. Em uma atribuição ou atribuição implícita em uma chamada de função, a origem e o destino devem ser da mesma classe de função, mas as classes de função não coincidem.
