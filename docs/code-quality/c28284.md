# <a name="c28284"></a>C28284
C28284 de aviso: Para símbolo, predicados atualmente não têm suporte para símbolos de não função

 Esse aviso indica que uma anotação condicional (predicado, `__drv_when`) foi encontrada em algo diferente de uma função.
