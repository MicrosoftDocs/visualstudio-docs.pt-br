# <a name="c6305"></a>C6305
Aviso C6305: potencial incompatível entre as quantidades de sizeof e countof

 Esse aviso indica que uma variável que contém um resultado `sizeof` está sendo adicionada ou subtraída de um ponteiro ou uma expressão `countof`. Isso causará um dimensionamento inesperado na aritmética do ponteiro.

## <a name="example"></a>Exemplo
 O código a seguir gera este aviso:

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Para corrigir esse aviso, use o seguinte código:

```
void f(int *p)
{
  // code...
  p += 1;
}
```
