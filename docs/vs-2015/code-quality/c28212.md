# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28212: expressão de ponteiro esperada para anotação  
  
 Este aviso indica que o parâmetro numerado em uma anotação (não a função que está sendo anotada) deve ser um ponteiro ou tipo de matriz, mas algum outro tipo foi encontrado. A anotação precisa ser corrigido.
