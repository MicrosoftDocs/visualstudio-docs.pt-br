# <a name="ca0056"></a>CA0056
Um arquivo de projeto tem uma versão incorreta da ferramenta de análise.

Esse aviso ocorre se FxCopCmd. exe for invocado manualmente com um arquivo de projeto que tem uma versão incorreta da ferramenta de análise de código.
