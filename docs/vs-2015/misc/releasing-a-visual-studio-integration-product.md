# <a name="releasing-a-visual-studio-integration-product"></a>Liberando um produto de integração do Visual Studio
Os tópicos nesta seção mostram como liberar seu produto de integração do Visual Studio criando e instalando o VSPackages e usando o VSIT (teste de integração do Visual Studio).

## <a name="in-this-section"></a>Nesta seção
 [Instalando o VSPackages](../misc/installing-vspackages.md) Discute como criar e instalar o VSPackages e como dar suporte a usuários que executam várias versões do Visual Studio ao mesmo tempo.

 [Utilitário RegPkg](../extensibility/internals/regpkg-utility.md) Descreve o utilitário RegPkg, que é usado para registrar um VSPackage e prepará-lo para implantação.

 [Guia do usuário e instalação do Premier Partner Edition](https://msdn.microsoft.com/8ee4dad7-95d3-4f2d-a8d4-3ba9a80ecae2) Fornece aos parceiros VSIP uma versão redistribuível do IDE (ambiente de desenvolvimento integrado) do Visual Studio para que eles possam adicionar suas próprias ferramentas de desenvolvimento e idiomas de computador.
