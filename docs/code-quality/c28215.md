# <a name="c28215"></a>C28215
C28215 de aviso: O typefix não pode ser aplicado a um parâmetro que já tem um typefix

 A aplicação de uma anotação `__typefix` a um parâmetro que já tem essa anotação é um erro. As anotações `__typefix` são usadas apenas em alguns casos especiais e esse aviso não deve ser visto no uso normal.
