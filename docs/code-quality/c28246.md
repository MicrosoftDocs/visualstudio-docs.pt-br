# <a name="c28246"></a>C28246
Aviso C28246: A anotação para função '\<nome >'-parâmetro '\<parâmetro >' não coincide com o tipo do parâmetro

 Um operador de deref foi aplicado a um tipo de ponteiro não durante a criação de uma anotação.
