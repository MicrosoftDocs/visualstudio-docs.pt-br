# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28302: para o parâmetro de referência do C++ <parameter_name>, um `_Deref_` operador extra foi encontrado em \<annotation> .  
  
 Esse aviso é relatado quando um nível extra de `_Deref_` é usado em um parâmetro.  
  
 O SAL2 não exige o uso de um nível extra de `_Deref_` ao lidar com parâmetros de referência. Essa anotação específica não é ambígua e é interpretada corretamente, mas deve ser corrigida.  
  
 Frequentemente, isso pode ser corrigido simplesmente removendo a `__deref` anotação mais antiga e usando a sintaxe SAL2. Às vezes, pode ser necessário usar `_At_` o para fazer referência ao objeto específico a ser anotado.
