# <a name="c28220"></a>C28220
C28220 de aviso: Expressão inteira esperada para anotação do parâmetro na anotação

 Esse aviso indica que um parâmetro para uma anotação deve ser uma expressão de número inteiro e algum outro tipo foi encontrado. Isso geralmente indica uma macro de anotação codificada incorretamente.
