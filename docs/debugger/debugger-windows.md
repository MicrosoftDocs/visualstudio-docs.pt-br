# <a name="inspect-data-using-debugger-windows-in-visual-studio"></a>Inspecionar dados usando janelas do depurador Visual Studio

Você pode abrir a maioria das janelas do depurador durante a depuração do programa. Para ver uma lista de janelas do depurador, depure um ponto de interrupção e inicie a depuração. Quando você atingir o ponto de interrupção e a execução parar, clique **em Depurar > Windows**.
