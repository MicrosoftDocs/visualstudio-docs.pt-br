# <a name="xml-schema-designer-overview"></a>Visão geral do Designer de esquema XML

O Designer de Esquema XML (Designer XSD) é uma ferramenta gráfica por meio da qual é possível visualizar um esquema definido em níveis distintos de abstração. Os principais componentes do Designer XSD são:

- [XML Schema Explorer](../xml-tools/xml-schema-explorer.md), que permite procurar e navegar na árvore de esquema XML e executar pesquisas.

- O [exibição inicial](../xml-tools/start-view.md), que é o ponto de partida para o Designer XSD. O modo de início, você pode navegar até outras exibições do Designer XSD e ver os detalhes de seu conjunto de esquema.

- O [modo de exibição gráfico](../xml-tools/graph-view.md), que permite que você veja uma visão geral de um conjunto de esquema e as relações entre os nós do esquema.

- O [modo de exibição do modelo de conteúdo](../xml-tools/content-model-view.md), que fornece uma representação gráfica dos detalhes de nós de esquema local e global, incluindo tipos complexos e simples, elementos, grupos, atributos e grupos de atributos.
