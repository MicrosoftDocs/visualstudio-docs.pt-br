# <a name="ca0502"></a>CA0502
Idioma sem suporte

Esse erro ocorre quando o usuário tenta analisar um projeto ASP.NET que contém uma ou mais páginas ou classes escritas em uma linguagem que não seja C# ou Visual Basic. Para corrigir esse erro, remova o arquivo do projeto ou desabilite a análise de código.
