# <a name="extending-properties-and-the-property-window"></a>Estendendo propriedades e a janela Propriedade
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A janela **Propriedades** é usada para exibir e editar as propriedades de tempo de design e os eventos dos objetos selecionados.  
  
 Quando você seleciona um objeto no [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] , as propriedades públicas desse objeto aparecem na janela **Propriedades** . Para obter mais informações, consulte [visão geral da janela Propriedades](../extensibility/internals/properties-window-overview.md).
