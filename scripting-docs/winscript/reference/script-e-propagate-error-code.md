# <a name="scriptepropagate-error-code"></a>Código de erro SCRIPT_E_PROPAGATE
Um erro de script está sendo propagado para o chamador, que pode estar em um thread diferente. O host deve passar o código de erro para o chamador.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
