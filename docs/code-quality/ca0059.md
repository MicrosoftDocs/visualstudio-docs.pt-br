# <a name="ca0059"></a>CA0059
Erro de opção de linha de comando.

Esse erro ocorrerá se uma opção de linha de comando necessária estiver ausente, por exemplo,

a opção/out ou/console não foi especificada. Esse erro também ocorrerá se houver uma opção inválida, como um valor de tempo limite incorreto.
