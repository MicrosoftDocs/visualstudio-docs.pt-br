# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28213: A `_Use_decl_annotations_` anotação deve ser usada para fazer referência, sem modificações, uma declaração prévia. \<Por que >  
  
 Esse aviso é relatado quando uma declaração anterior é referenciada sem necessários `_Use_decl_annotations_` anotação.
