# <a name="c26101"></a>C26101
C26101 de aviso: Falha ao usar a operação Interlocked corretamente para a variável \<var >.

 As APIs do Windows oferecem uma variedade de operações interbloqueadas. A anotação `_Interlocked_` especifica que uma variável só deve ser acessada por meio de uma operação intercadeado. Aviso C26101 é emitido quando um acesso não é consistente com a anotação `_Interlocked_`.

## <a name="example"></a>Exemplo
 O exemplo a seguir gera o aviso C26101 porque há uma violação do contrato `_Interlocked_`.

```

CRITICAL_SECTION cs;
typedef struct _DATA
{
    _Interlocked_ LONG data;
} DATA;

void Safe(DATA* p)
{
    InterlockedIncrement(&p->data); // OK
}

void Unsafe(DATA* p)
{
    p->data += 1; // Warning C26101
    EnterCriticalSection(&cs);
    LeaveCriticalSection(&cs);
}
```
