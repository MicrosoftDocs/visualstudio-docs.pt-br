# <a name="ca0504"></a>CA0504

O diretório de regras foi ignorado porque não existe.

Aviso CA504 é gerado quando um diretório especificado na propriedade **CodeAnalysisRuleDirectories** em um projeto ou arquivo de configuração de compilação não foi encontrado. Para resolver esse problema, insira um caminho de diretório válido.
