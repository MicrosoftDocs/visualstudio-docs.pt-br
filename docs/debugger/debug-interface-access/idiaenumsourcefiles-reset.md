# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset

## <a name="syntax"></a>Sintaxe

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Valor retornado
 Retorna `S_OK`.
