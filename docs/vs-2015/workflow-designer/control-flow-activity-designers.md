# <a name="control-flow-activity-designers"></a>Designer de atividade de fluxo de controle
[!INCLUDE[wfd1](../includes/wfd1-md.md)] inclui uma série de atividades fornecidas pelo sistema que você pode usar ao construir seus fluxos de trabalho. Esta seção contém sistema forneceu as atividades usadas para controlar o fluxo em um fluxo de trabalho. Os tópicos a seguir descrevem essas atividades e fornecem orientação sobre como usá-los.

## <a name="in-this-section"></a>Nesta seção
 [DoWhile](../workflow-designer/dowhile-activity-designer.md) Executa a atividade contida em seu corpo pelo menos uma vez, até que uma condição especificada seja avaliada como **true**.

 [ForEach \<T> ](foreach-t-activity-designer.md) Executa a atividade contida em seu corpo para cada item em uma coleção especificada.

 [Se](../workflow-designer/if-activity-designer.md) Avalia uma condição e executa uma atividade dependendo dos resultados dessa avaliação.

 [Paralelo](../workflow-designer/parallel-activity-designer.md) Executa uma coleção de atividades filho simultaneamente.

 [ParallelForEach \<T> ](../workflow-designer/parallelforeach-t-activity-designer.md) Enumera os elementos de uma coleção e executa uma instrução inserida para cada elemento da coleção em paralelo

 [Escolha](../workflow-designer/pick-activity-designer.md) Executa uma de várias ramificações em resposta a algum evento que fornece controle de fluxo baseado em eventos.

 [PickBranch](../workflow-designer/pickbranch-activity-designer.md) Fornece um caminho potencial de execução em uma <xref:System.Activities.Statements.Pick> atividade.

 [Sequência](../workflow-designer/sequence-activity-designer.md) de Contém uma coleção ordenada de atividades filho que ele executa na ordem.

 [Alternar \<T> ](switch-t-activity-designer.md) Avalia uma expressão especificada e executa a atividade de uma coleção de atividades cuja chave associada corresponde ao valor obtido da avaliação.

 [Enquanto](../workflow-designer/while-activity-designer.md) Executa a atividade contida em seu corpo enquanto uma condição especificada é avaliada como **true**.
