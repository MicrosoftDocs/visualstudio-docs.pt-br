# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28208: Função \<função > foi definida anteriormente com uma lista de parâmetros diferentes no \<arquivo > (\<linha >). Algumas ferramentas de análise produzirão resultados incorretos  
  
 Esse aviso é relatado quando a definição de uma função conhecidos não corresponde a outra ocorrência.
