# <a name="c28022"></a>C28022
C28022 de aviso: As classes de função \<classlist1 > nesta função não correspondem às classes de função \<classlist2 > no typedef usado para defini-la.

 Esse aviso é relatado quando há um erro nas anotações. O typedef e a própria função têm anotações `_Function_class_`, mas não coincidem. Se ambos forem usados, eles deverão corresponder.
