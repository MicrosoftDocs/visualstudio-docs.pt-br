# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
Redefine uma sequência de enumeração para o início.

## <a name="syntax"></a>Sintaxe

```
HRESULT Reset();
```

## <a name="return-value"></a>Valor de retorno
 Retorna `S_OK`.
