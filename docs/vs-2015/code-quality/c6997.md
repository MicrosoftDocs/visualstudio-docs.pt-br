# <a name="c6997"></a>C6997
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso 6997: As anotações neste local não fazem sentidas e serão ignoradas.  
  
 As anotações não podem ser aplicadas a `extern "C" {...}`. Aplique as anotações a um objeto específico.
