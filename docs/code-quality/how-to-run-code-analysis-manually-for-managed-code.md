# <a name="how-to-run-code-analysis-manually-for-managed-code"></a>Como executar a análise de código manualmente para código gerenciado
A ferramenta de análise de código fornece informações a você sobre possíveis defeitos no seu código-fonte. Você pode executar a análise de código automaticamente cada versão de um projeto de código, e você também pode executar a análise de código manualmente. As regras que são verificadas quando a análise de código é executada são especificadas na página de análise de código das páginas de propriedades do projeto. Para obter mais informações, consulte [como: configurar a análise de código para um projeto de código gerenciado](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)

### <a name="to-run-code-analysis-manually"></a>Para executar a análise de código manualmente

1.  Em **Solution Explorer**, clique no projeto.

2.  Sobre o **analisar** menu, clique em **executar análise de código em** *nome do projeto*.
