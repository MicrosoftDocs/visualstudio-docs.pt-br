# <a name="live-unit-testing-overview"></a>Visão geral do teste da unidade ao vivo

O Live Unit Testing executa os testes da unidade automaticamente e em tempo real à medida que você faz alterações de código. Isso permite que você refatore e mude o código com maior confiança. O Live Unit Testing executa automaticamente todos os testes impactados à medida que você edita seu código para garantir que suas alterações não introduzam regressões.

O teste da unidade ao vivo indica se os testes da unidade cobrem adequadamente o seu código. Ele retrata graficamente a cobertura de código em tempo real. Você pode ver rapidamente quantos testes cobrem cada linha de código e quais linhas não são cobertas por nenhum teste de unidade.

Se você tiver uma solução que inclua um ou mais projetos de teste de unidade, você pode habilitar o teste da unidade ao vivo selecionando **Test** > **Live Unit Testing** > **Start** a partir da barra de menu de nível superior no Visual Studio.

> [!NOTE]
> O Teste de Unidade Ao Vivo só está disponível na edição visual studio enterprise.
