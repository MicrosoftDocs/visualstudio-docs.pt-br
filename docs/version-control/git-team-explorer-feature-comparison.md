# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Comparação lado a lado do Git e do Team Explorer

Lançamos a primeira versão de uma nova experiência do Git no Visual Studio 2019 [versão 16.8](/visualstudio/releases/2019/release-notes/). Essa nova experiência ajuda a reduzir a alternação de contexto com uma janela **simples alterações** do Git que inclui tarefas comuns do Git. Ele também inclui uma janela de repositório **Git** em toda a tela para operações mais avançadas do Git, como gerenciamento de branch e navegação no repositório.

Se você estiver usando o Team Explorer, este é um guia passo a passo que explica como você pode usar a nova experiência do Git.

> [!NOTE]
> A seção a seguir inclui capturas de tela dimensionada para caber nas colunas de uma tabela. Clique em cada captura de tela para exibir uma versão maior dela. (Se você estiver usando um dispositivo touchscreen, toque em cada captura de tela para exibir uma versão maior dele.)
