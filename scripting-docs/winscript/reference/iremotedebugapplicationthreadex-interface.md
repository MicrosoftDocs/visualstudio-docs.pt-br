# <a name="iremotedebugapplicationthreadex-interface"></a>Interface IRemoteDebugApplicationThreadEx
Representa um thread de execução dentro de um aplicativo específico.  
  
 Além dos métodos herdados de `IUnknown`, o `IRemoteDebugApplicationThreadEx` interface expõe os métodos a seguir.  
  
## <a name="methods-in-vtable-order"></a>Métodos na ordem de Vtable  
  
|Método|Descrição|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|Enumera os contextos de expressão global para todos os idiomas em execução neste thread.|
