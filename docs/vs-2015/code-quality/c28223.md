# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28223: não foi possível encontrar token/identificador esperado para a anotação, parâmetro  
  
 Esse aviso indica que um parâmetro para uma anotação deve ser um identificador e algum outro símbolo foi encontrado. Isso geralmente indica uma macro de anotação codificada incorretamente. O uso de C ou C++ palavras-chave nesta posição causará esse erro.
