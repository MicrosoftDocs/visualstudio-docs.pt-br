# <a name="how-to-create-a-work-item-for-a-managed-code-defect"></a>Como criar um item de trabalho para um defeito de código gerenciado
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Você pode usar o recurso de acompanhamento de item de trabalho para registrar o item de trabalho no [!INCLUDE[vsPreLong](../includes/vsprelong-md.md)]. Para usar esse recurso, seu projeto deve fazer parte do projeto de equipe no [!INCLUDE[esprfound](../includes/esprfound-md.md)].

### <a name="to-create-a-work-item-for-managed-code-defect"></a>Para criar um item de trabalho para um defeito de código gerenciado

1. Na janela **análise de código** , selecione o aviso.

2. Escolha **ações**, escolha **Criar item de trabalho** e escolha o tipo de item de trabalho a ser criado.

     Um novo item de trabalho é criado para você especificar as informações de defeito.

### <a name="to-create-a-work-item-for-multiple-managed-code-defects"></a>Para criar um item de trabalho para vários defeitos de código gerenciado

1. Na **lista de erros**, selecione vários avisos e clique com o botão direito do mouse nos avisos.

2. Aponte para **Criar item de trabalho** e clique no tipo de item de trabalho a ser criado.

     Um único item de trabalho é criado para todos os avisos selecionados para que você especifique as informações do bug.
