# <a name="image-service-tools"></a>Ferramentas de serviço de imagem
O VS SDK inclui várias ferramentas para ajudar os construtores de extensão a usar o serviço de imagem Visual Studio.

- A ferramenta [Manifest from Resources](../../extensibility/internals/manifest-from-resources.md) pega uma lista de recursos de imagem (.png ou arquivos .xaml) e gera um arquivo manifesto de imagem.

- A ferramenta [Manifest to Code](../../extensibility/internals/manifest-to-code.md) pega um arquivo manifesto de imagem e gera um arquivo de invólucro para usar em arquivos C++, C#, VB ou .vsct.

- O [Visualizador da Biblioteca de Imagens](../../extensibility/internals/image-library-viewer.md) pode carregar, editar e pesquisar manifestos de imagem.
