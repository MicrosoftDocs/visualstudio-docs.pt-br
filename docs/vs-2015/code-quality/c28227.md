# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28227: a anotação também não pode ser um PrimOp (consulte a declaração anterior)  
  
 Esse aviso indica que a anotação nomeada está sendo declarada como uma anotação comum e também foi declarada anteriormente como um PrimOp. Isso não deve ser possível se as macros padrão estiverem sendo usadas para anotações; Esse aviso não deve ser visto no uso típico.
