# <a name="expected--javascript"></a>')' esperado (JavaScript)
Você tentou colocar uma expressão dentro de um conjunto de parênteses, mas não incluiu o parêntese de fechamento. Algumas expressões devem ser colocadas dentro de um conjunto de parênteses de abertura e fechamento. Observe o uso de parênteses no exemplo a seguir.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione os parênteses à direita à expressão de avaliação.
