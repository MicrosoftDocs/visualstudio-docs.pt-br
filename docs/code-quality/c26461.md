# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  O argumento de ponteiro '% Argument% ' para a função '% function% ' pode ser marcado como um ponteiro para `const`. Ver [con.3 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
