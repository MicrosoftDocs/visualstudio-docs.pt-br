# <a name="symbol-load-information-dialog-box"></a>Caixa de diálogo Informações de Carga de Símbolo
A caixa de diálogo **Informações de Carregamento de Símbolos** é aberta quando você clica com o botão direito do mouse na janela **Módulos** e escolhe **Informações de Carregamento de Símbolos**.

## <a name="uielement-list"></a>Lista de elementos de interface do usuário
 **Informações** do Mostra os diretórios pesquisados para arquivos de símbolo (arquivos PDB) com base nas configurações de instalação do Visual Studio e nas alterações feitas nas configurações de símbolo na caixa de diálogo **Opções** .

 **Configurações de símbolo** Clique neste botão para alterar as configurações de símbolo na caixa de diálogo **Opções** (categoria **depuração** , página **símbolos** ). Nessa página, você pode adicionar ou remover locais de símbolos, especificar uma localização de cache local se estiver usando um servidor de símbolos e controlar quando locais de símbolos forem pesquisados.
