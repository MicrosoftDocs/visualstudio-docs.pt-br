# <a name="font-dialog-box-microsoft-spy-help"></a>Caixa de diálogo Fonte (Ajuda do Microsoft Spy++)
Use essa caixa de diálogo para configurar a fonte de caracteres usada em uma janela de [modos de exibição do Spy + +](../debugger/spy-increment-views.md) . Para exibir essa caixa de diálogo, expanda o menu **Exibir** e escolha **fonte**.

 As seguintes configurações estão disponíveis:

 **Fonte** A fonte de caracteres para a janela de modos de exibição do Spy + + que tem foco.

 **Estilo da fonte** O estilo da fonte para a janela de modos de exibição do Spy + + que tem foco.

 **Tamanho** O tamanho da fonte para a janela de modos de exibição do Spy + + que tem foco.

 **Salvar fonte como padrão** Salva as configurações de fonte atuais como suas preferências ao abrir as exibições do Spy + +.
