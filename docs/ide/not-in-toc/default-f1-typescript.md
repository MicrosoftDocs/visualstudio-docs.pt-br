# <a name="typescript-f1-help-for-visual-studio"></a>Ajuda F1 do TypeScript para Visual Studio

O TypeScript é uma linguagem que é um superconjunto tipado do JavaScript; Portanto, a sintaxe do JavaScript é o TypeScript legal. O TypeScript adiciona regras sobre como tipos diferentes de valores podem ser usados.

Para obter ajuda de referência sobre palavras-chave de idioma e elementos de código, consulte:

- [Documentação do typescript](https://www.typescriptlang.org/docs) para obter ajuda com o typescript.
- [MDN Web docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) para obter ajuda com o JavaScript. Na caixa de pesquisa, digite o nome da palavra-chave que você está procurando.
- [Página de pesquisa](/search) em docs.Microsoft.com se você estiver procurando ajuda na documentação oficial da Microsoft.
