# <a name="idiaenuminjectedsourcesreset"></a>IDiaEnumInjectedSources::Reset

## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Retorna `S_OK`.
