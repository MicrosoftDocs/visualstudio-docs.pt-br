# <a name="ca0056"></a>CA0056
Um arquivo de projeto tem uma versão incorreta do que a ferramenta de análise.

 Este aviso ocorre se FxCopCmd.exe é invocado manualmente com um arquivo de projeto que tenha uma versão de ferramenta de análise de código incorreto.
