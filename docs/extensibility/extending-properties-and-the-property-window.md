# <a name="extend-properties-and-the-property-window"></a>Estender Propriedades e a janela de propriedades
A janela **Propriedades** é usada para exibir e editar as propriedades de tempo de design e os eventos dos objetos selecionados.

 Quando você seleciona um objeto no [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] , as propriedades públicas desse objeto aparecem na janela **Propriedades** . Para obter mais informações, consulte [janela Propriedades visão geral](../extensibility/internals/properties-window-overview.md).
