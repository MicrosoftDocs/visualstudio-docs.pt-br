# <a name="c28244"></a>C28244
C28244 de aviso: A anotação for function tem um parâmetro/anotação externa não analisável

 No momento, isso não deve ser possível se as macros padrão estiverem sendo usadas para anotações; Esse aviso não deve ser visto no uso típico.
