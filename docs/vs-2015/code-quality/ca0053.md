# <a name="ca0053"></a>CA0053

Não foi possível carregar o assembly da regra.

Isso indica que um assembly de regra não pôde ser carregado por reflexão. Isso pode ocorrer se o usuário tiver regras personalizadas que não foram criadas corretamente ou se a ferramenta de análise não puder carregar o assembly por outros motivos, como o acesso ao sistema de arquivos foi negado.
