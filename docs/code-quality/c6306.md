# <a name="c6306"></a>C6306
Aviso C6306: chamada incorreta para o \<function >: considere o uso de \<Function > que aceita um va_list como um argumento

 Esse aviso indica uma chamada de função incorreta. A família `printf` inclui várias funções que usam uma lista variável de argumentos; no entanto, essas funções não podem ser chamadas com um argumento `va_list`. Há uma família de funções `vprintf` correspondente que pode ser usada para essas chamadas. Chamar a função de impressão errada causará saída incorreta.

## <a name="example"></a>Exemplo
 O código a seguir gera este aviso:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 Para corrigir esse aviso, use o seguinte código:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
