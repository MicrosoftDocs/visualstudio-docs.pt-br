# <a name="ca2227-collection-properties-should-be-read-only"></a>CA2227: as propriedades de coleção devem ser somente leitura

|||
|-|-|
|TypeName|CollectionPropertiesShouldBeReadOnly|
|CheckId|CA2227|
|Categoria|Microsoft.Uso|
|Mudança de ruptura|Quebra|

## <a name="cause"></a>Causa

Uma propriedade externamente visível e gravável é <xref:System.Collections.ICollection?displayProperty=fullName>de um tipo que implementa . Essa regra ignora matrizes, indexadores (propriedades com o nome 'Item'), coleções imutáveis, coleções somente leituras e conjuntos de permissões.

## <a name="rule-description"></a>Descrição da regra

Uma propriedade de coleção gravável permite que o usuário substitua a coleção por uma coleção completamente diferente. Uma propriedade somente leitura impede que a coleção seja substituída, mas ainda permite que os membros individuais sejam definidos. Se substituir a coleção for um objetivo, o padrão de design preferido é incluir um método para remover todos os elementos da coleção e um método para repovoar a coleção. Veja <xref:System.Collections.ArrayList.Clear%2A> os <xref:System.Collections.ArrayList.AddRange%2A> métodos <xref:System.Collections.ArrayList?displayProperty=fullName> e métodos da classe para um exemplo deste padrão.

Tanto a serialização binária quanto a serialização XML suportam propriedades somente leitura que são coleções. A <xref:System.Xml.Serialization.XmlSerializer?displayProperty=fullName> classe tem requisitos <xref:System.Collections.ICollection> específicos <xref:System.Collections.IEnumerable?displayProperty=fullName> para tipos que implementam e para serem serializáveis.

## <a name="how-to-fix-violations"></a>Como corrigir violações

Para corrigir uma violação desta regra, faça a propriedade somente leitura. Se o projeto exigir, adicione métodos para limpar e repovoar a coleção.

## <a name="when-to-suppress-warnings"></a>Quando suprimir avisos

Você pode suprimir o aviso se a propriedade fizer parte de uma classe [DTO (Data Transfer Object, objeto de transferência](/previous-versions/msp-n-p/ff649585(v=pandp.10)) de dados).

Caso contrário, não suprima avisos desta regra.
