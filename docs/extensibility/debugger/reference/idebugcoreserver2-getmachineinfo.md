# <a name="idebugcoreserver2getmachineinfo"></a>IDebugCoreServer2::GetMachineInfo
Recupera uma descrição da máquina em que o servidor principal está sendo executado.

## <a name="syntax"></a>Sintaxe

```cpp
HRESULT GetMachineInfo( 
   MACHINE_INFO_FIELDS Fields,
   MACHINE_INFO*       pMachineInfo
);
```

```csharp
int GetMachineInfo( 
   enum_ MACHINE_INFO_FIELDS  Fields,
   MACHINE_INFO[]             pMachineInfo
);
```

## <a name="parameters"></a>parâmetros
`Fields`\
[em] Uma combinação de [MACHINE_INFO_FIELDS](../../../extensibility/debugger/reference/machine-info-fields.md) bandeiras da enumeração `pMachineInfo` MACHINE_INFO_FIELDS que especificam quais campos devem ser preenchidos.

 `pMachineInfo`\

 [dentro, fora] Uma [estrutura MACHINE_INFO](../../../extensibility/debugger/reference/machine-info.md) que é preenchida com uma descrição da máquina.

## <a name="return-value"></a>Valor retornado
 Se for `S_OK`bem sucedido, retorna; caso contrário, retorna um código de erro.
