# <a name="c28223"></a>C28223
Aviso C28223: Não localizou o Token/identificador esperado para anotação, parâmetro

 Esse aviso indica que um parâmetro em uma anotação deve ser um identificador, e algum outro símbolo foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente. O uso de palavras-chave C ou C++ nessa posição fará com que esse erro.
