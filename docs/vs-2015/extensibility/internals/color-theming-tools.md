# <a name="color-theming-tools"></a>Ferramentas de temas de cores
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

O SDK do VS fornece várias ferramentas para ajudá-lo a criar e gerenciar temas de cores.  
  
- A ferramenta [Editor de cores do VSIX](../../extensibility/internals/vsix-color-editor.md) pode criar e Editar cores personalizadas para o Visual Studio.  
  
- A ferramenta do [compilador de cores do VSIX](../../extensibility/internals/vsix-color-compiler.md) converte arquivos. XML de tema do Visual Studio existentes em. pkgdef para sua extensão.
