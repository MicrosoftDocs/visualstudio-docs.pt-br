# <a name="c28163"></a>C28163
C28163 de aviso: A função nunca deve ser chamada de dentro de um bloco try/Except

 Esse aviso é relatado quando uma função é de um tipo que nunca deve ser colocado entre um bloco `try/except` é encontrado em um bloco `try/except`. A ferramenta de análise de código encontrou pelo menos um caminho no qual a função chamada estava dentro de um bloco `try/except`.
