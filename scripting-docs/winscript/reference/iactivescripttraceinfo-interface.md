# <a name="iactivescripttraceinfo-interface"></a>Interface IActiveScriptTraceInfo
Fornece métodos para iniciar e parar o rastreamento.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[IActiveScriptTraceInfo::StartScriptTracing Method](../../winscript/reference/iactivescripttraceinfo-startscripttracing-method.md)|Inicia o rastreamento de script.|  
|[IActiveScriptTraceInfo::StopScriptTracing Method](../../winscript/reference/iactivescripttraceinfo-stopscripttracing-method.md)|Interrompe o rastreamento.|
