# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28224: anotação requer parâmetros  
  
 Este aviso indica que a anotação nomeada for usada sem parâmetros e pelo menos um parâmetro é necessário. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
