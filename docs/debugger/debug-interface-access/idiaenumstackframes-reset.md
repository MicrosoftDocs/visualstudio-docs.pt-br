# <a name="idiaenumstackframesreset"></a>IDiaEnumStackFrames::Reset
Redefine a sequência de enumeração para o início.

## <a name="syntax"></a>Sintaxe

```C++
HRESULT Reset();
```

## <a name="return-value"></a>Valor retornado
 Retorna `S_OK`.
