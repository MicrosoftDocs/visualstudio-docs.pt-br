# <a name="how-to-display-spy-views"></a>Como mostrar exibições do Spy++
Quando o Spy + + é iniciado, ele é aberto no [modo de exibição do Windows](../debugger/windows-view.md), um modo de exibição de árvore de todas as janelas e controles disponíveis em seu computador. Esta janela raiz é legendada "Windows 1". Há também três outras exibições do Spy + + disponíveis: exibição de [mensagens](../debugger/messages-view.md), [exibição de processos](../debugger/processes-view.md)e [exibição de threads](../debugger/threads-view.md). Você pode exibir a exibição desejada no menu do **Spy** . Para obter mais informações, consulte [modos de exibição do Spy + +](../debugger/spy-increment-views.md).
