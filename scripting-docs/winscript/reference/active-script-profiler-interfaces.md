# <a name="active-script-profiler-interfaces"></a>Interfaces do criador de perfil de script ativo
As interfaces do criador de perfil de script ativo permitem que você receba eventos de criação de perfil do mecanismo de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
 O arquivo de cabeçalho activprof. h fornece as interfaces listadas nesta seção.  
  
## <a name="in-this-section"></a>Nesta seção  
 As interfaces a seguir habilitam a criação de perfil:  
  
- [Interface IActiveScriptProfilerControl](../../winscript/reference/iactivescriptprofilercontrol-interface.md)  
  
- [Interface IActiveScriptProfilerControl2](../../winscript/reference/iactivescriptprofilercontrol2-interface.md)  
  
- [Interface IActiveScriptProfilerControl3](../../winscript/reference/iactivescriptprofilercontrol3-interface.md)  
  
- [Interface IActiveScriptProfilerControl5](../../winscript/reference/iactivescriptprofilercontrol5-interface.md)  
  
- [Interface IActiveScriptProfilerCallback](../../winscript/reference/iactivescriptprofilercallback-interface.md)  
  
- [Interface IActiveScriptProfilerCallback2](../../winscript/reference/iactivescriptprofilercallback2-interface.md)  
  
- [Interface IActiveScriptProfilerCallback3](../../winscript/reference/iactivescriptprofilercallback3-interface.md)  
  
- [Interface IActiveScriptProfilerHeapEnum](../../winscript/reference/iactivescriptprofilerheapenum-interface.md)  
  
  A seção a seguir lista as enumerações que são usadas para a criação de perfil:  
  
- [Constantes, enumerações e estruturas de criador de perfil do script ativo](../../winscript/reference/active-script-profiler-constants-enumerations-and-structures.md)  
  
> [!NOTE]
> As interfaces do criador de perfil de script ativo foram lançadas primeiro com o Internet Explorer 8. As interfaces `IActiveScriptProfilerControl2` e `IActiveScriptProfilerCallback2` foram lançadas pela primeira vez com o Internet Explorer 9. A [interface IActiveScriptProfilerControl3](../../winscript/reference/iactivescriptprofilercontrol3-interface.md), a [interface IActiveScriptProfilerCallback3](../../winscript/reference/iactivescriptprofilercallback3-interface.md)e as interfaces de [interface IActiveScriptProfilerHeapEnum](../../winscript/reference/iactivescriptprofilerheapenum-interface.md) foram lançadas pela primeira vez com o Internet Explorer 10. A [interface IActiveScriptProfilerControl5](../../winscript/reference/iactivescriptprofilercontrol5-interface.md) foi lançada pela primeira vez com o Internet Explorer 11.  
>   
> No Internet Explorer 8 e no Internet Explorer 9, somente a linguagem [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] usa essas interfaces para dar suporte à criação de perfil de script.
