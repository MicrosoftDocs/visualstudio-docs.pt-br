# <a name="overview-of-code-analysis-for-managed-code-in-visual-studio"></a>Visão geral da análise de código para código gerenciado no Visual Studio

O Visual Studio pode executar a análise de código do código gerenciado de duas maneiras: com a [análise herdada](../code-quality/walkthrough-analyzing-managed-code-for-code-defects.md), também conhecida como análise estática do FxCop de assemblies gerenciados e com os mais modernos analisadores de [código](../code-quality/roslyn-analyzers-overview.md)baseados em .net Compiler Platform. Analisadores de código baseados em .NET Compiler Platform, que analisam seu código ao vivo conforme você digita, substituem a análise de código estático do FxCop herdado, que apenas analisa o código compilado.
