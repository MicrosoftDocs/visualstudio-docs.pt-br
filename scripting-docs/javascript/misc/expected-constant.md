# <a name="expected-constant"></a>Constante esperada
Você tentou usar uma variável (compilação de condicional não) em uma instrução de teste de compilação condicional. Instruções de teste de compilação condicional devem ser avaliada como uma constante.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Substitua a variável com um valor literal.  
  
-   Substitua a variável com uma variável de compilação condicional.
