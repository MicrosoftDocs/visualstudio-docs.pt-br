# <a name="c28219"></a>C28219
aviso C28219: membro de enumeração esperado para a anotação do parâmetro na anotação

 Um parâmetro para uma anotação deve ser um membro do tipo de `enum` nomeado e algum outro símbolo foi encontrado; Use um membro desse tipo de `enum`. Isso geralmente indica uma macro de anotação codificada incorretamente.
