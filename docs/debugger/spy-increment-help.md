# <a name="spy-help"></a>Ajuda do Spy++
A documentação para o utilitário Spy + + inclui dois grupos de tópicos gerais em usando Spy + + e sobre as exibições do Spy + +. Elas são seguidas por Spy + + referência, que fornece mais detalhes sobre cada caixa de caixa de diálogo e o comando de menu disponíveis.

## <a name="using-spy"></a>Usando Spy++
 [Usando Spy + +](../debugger/using-spy-increment.md) apresenta a ferramenta Spy + + e explica como ele pode ser usado.

 [Exibições do Spy + +](../debugger/spy-increment-views.md) explica as exibições de árvore do Spy + + do windows, as mensagens, processos e threads.

## <a name="spy-reference"></a>Referência de Spy++
 [Comandos de menu](../debugger/menu-commands.md) descreve os comandos disponíveis em cada menu do Spy + +.

 [Caixa de diálogo fonte](../debugger/font-dialog-box-microsoft-spy-increment-help.md) usado para configurar a fonte de caracteres usada em uma janela de exibições do Spy + +.

 [Ferramentas de pesquisa para exibições do Spy + +](../debugger/search-tools-for-spy-increment-views.md) caixas de diálogo usadas para localizar um nó específico em uma janela de exibições do Spy + +.

 [Caixa de diálogo de propriedades de janela](../debugger/window-properties-dialog-box.md) usado para exibir as propriedades de uma janela selecionada no modo de exibição do Windows.

 [Caixa de diálogo Opções da mensagem](../debugger/message-options-dialog-box.md) usado para selecionar quais mensagens são listadas na exibição de mensagens.

 [Caixa de diálogo Propriedades da mensagem](../debugger/message-properties-dialog-box.md) usado para exibir as propriedades de uma mensagem selecionada na exibição de mensagens.

 [Caixa de diálogo Propriedades de processar](../debugger/process-properties-dialog-box.md) usado para exibir as propriedades de um processo selecionado na exibição de processos.

 [Caixa de diálogo Propriedades do thread](../debugger/thread-properties-dialog-box.md) usado para exibir as propriedades de um thread selecionado na exibição de Threads.

 [Outras caixas de diálogo](../debugger/other-dialog-boxes.md) usado para exibir informações sobre o programa Spy + +.
