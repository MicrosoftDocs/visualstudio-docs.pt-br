# <a name="ca0505"></a>CA0505

A propriedade preterida será ignorada porque a propriedade de dioblíquo está definida.

Aviso CA505 é gerado quando as propriedades de análise de código preteridas e substituídas contêm valores em um projeto ou arquivo de configuração de compilação. Para resolver esse problema, remova a propriedade preterida.
