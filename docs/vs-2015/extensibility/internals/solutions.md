# <a name="solutions"></a>Soluções
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Uma solução é um agrupamento de um ou mais projetos que trabalham juntos para criar um aplicativo. As informações de projeto e status que pertencem à solução são armazenadas em dois arquivos de solução, um arquivo de solução baseada em texto (. sln) e um arquivo de opção de solução binária (. suo).
