# <a name="c28273"></a>C28273
warning C28273: Para função, os indícios são inconsistentes com a declaração da função

 Este aviso indica um erro nas anotações, não no código que está sendo analisado. As anotações que aparecem em uma definição de função são inconsistentes com aqueles que aparecem em uma declaração. As duas anotações devem ser resolvidas para corresponder.
