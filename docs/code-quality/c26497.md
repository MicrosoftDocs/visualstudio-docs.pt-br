# <a name="c26497-use_constexpr_for_function"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

Esta função% function% poderá ser marcada `constexpr` se a avaliação do tempo de compilação for desejada. Ver [diretrizes principais do C++ F.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr).
