# <a name="how-to-display-window-properties"></a>Como exibir Propriedades da Janela
Para saber mais sobre uma janela listada no [modo de exibição do Windows](../debugger/windows-view.md), use a [caixa de diálogo Propriedades da janela](../debugger/window-properties-dialog-box.md).

### <a name="to-open-the-window-properties-dialog-box"></a>Para abrir a caixa de diálogo Propriedades da janela

1. Selecione o item e, em seguida, escolha **Propriedades** no menu **Exibir** .

   As caixas de diálogo de propriedades não são modais. À medida que você clica de item para item em uma janela de exibição, a caixa de diálogo é atualizada automaticamente com informações em cada item selecionado.
