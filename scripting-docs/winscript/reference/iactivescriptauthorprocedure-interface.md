# <a name="iactivescriptauthorprocedure-interface"></a>Interface IActiveScriptAuthorProcedure
Além dos métodos herdados de `IUnknown`, a interface `IScriptAuthorProcedure` expõe o método a seguir.  
  
## <a name="methods-in-vtable-order"></a>Métodos em ordem vtable  
  
|Método|Descrição|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analisa um procedimento de código, adiciona o texto do procedimento de código ao mecanismo de criação de script e cria um objeto `IScriptEntry` que corresponde ao procedimento de código.|
