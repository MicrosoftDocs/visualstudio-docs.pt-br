## <a name="windows-developer-technical-support"></a>Suporte técnico para desenvolvedores do Windows  

O suporte a assinaturas do Visual Studio fornece assistência com vendas, assinaturas, contas e cobrança para assinaturas do Visual Studio. Se você tiver um problema de conta técnica ou de desenvolvedor do Windows, sua pergunta precisará ser direcionada para o [suporte do desenvolvedor do Windows](https://developer.microsoft.com/windows/support/?utm_source=developermscom). Sugerimos que você experimente esses outros recursos primeiro. 

## <a name="windows-technical-documentation"></a>Documentação técnica do Windows 
Examine os tutoriais, os guias de solução de problemas, os recursos úteis e uma variedade de outros tópicos na [documentação técnica do Windows](https://docs.microsoft.com/windows/).

## <a name="windows-developer-account"></a>Conta de desenvolvedor do Windows
Sua assinatura do Visual Studio pode incluir uma conta de desenvolvedor do Windows. Ative sua conta visitando a [página de benefícios do portal de assinaturas](https://my.visualstudio.com/Benefits) e clicando no bloco de benefícios. 

Leia mais sobre o [benefício da conta de desenvolvedor do Windows](https://docs.microsoft.com/visualstudio/subscriptions/vs-windows-dev). 

[Compare os benefícios da assinatura do Visual Studio](https://visualstudio.microsoft.com/vs/benefits/#azure?cat=visual-studio-enterprise-subscription) para descobrir o que está incluído na sua assinatura.
