# <a name="stepkind"></a>STEPKIND
Especifica o tipo de passo para pisar.

## <a name="syntax"></a>Sintaxe

```cpp
enum enum_STEPKIND { 
   STEP_INTO      = 0,
   STEP_OVER      = 1,
   STEP_OUT       = 2,
   STEP_BACKWARDS = 3
};
typedef DWORD STEPKIND;
```

```csharp
public enum enum_STEPKIND { 
   STEP_INTO      = 0,
   STEP_OVER      = 1,
   STEP_OUT       = 2,
   STEP_BACKWARDS = 3
};
```

## <a name="fields"></a>Campos
 `STEP_INTO`\
 Entra em uma função.

 `STEP_OVER`\
 Passa por cima de uma função.

 `STEP_OUT`\
 Sai de uma função.

 `STEP_BACKWARDS`\
 Desadeipara trás em uma função.

## <a name="remarks"></a>Comentários
 Passou como um argumento para o método [Step.](../../../extensibility/debugger/reference/idebugprocess3-step.md)
