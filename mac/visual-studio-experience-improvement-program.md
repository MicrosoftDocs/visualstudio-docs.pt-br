# <a name="visual-studio-customer-experience-improvement-program"></a>Programa de Aperfeiçoamento da Experiência do Usuário do Visual Studio

O VSCEIP (Programa de Aperfeiçoamento da Experiência do Usuário do Visual Studio) é um programa criado para ajudar a Microsoft a aperfeiçoar o Visual Studio para Mac ao longo do tempo. Esse programa coleta informações sobre o hardware do computador e como as pessoas usam o Visual Studio para Mac sem interromper os usuários em suas tarefas no computador. As informações coletadas ajudam a Microsoft a identificar quais funcionalidades devem ser aprimoradas. Este artigo descreve como aceitar ou recusar a VSCEIP.

Para obter mais informações sobre as informações coletadas, processadas ou transmitidas pelo VSCEIP, confira a [Política de privacidade da Microsoft](https://privacy.microsoft.com/privacystatement).

## <a name="choice-and-control-over-the-visual-studio-customer-experience-improvement-program"></a>Escolher e controlar o Programa de Aperfeiçoamento da Experiência do Usuário do Visual Studio

O VSCEIP está ativado por padrão. É possível desligá-lo ou ativá-lo novamente seguindo as instruções abaixo.

1. Abra o Visual Studio para Mac.

1. Na barra de menus, escolha **Visual Studio > Preferências > Outros > Comentários**.

1. Selecione sua preferência de participação:

    ![Selecione um botão de opção para refletir a sua preferência de participação](media/visual-studio-experience-improvement-program-image1.png)
