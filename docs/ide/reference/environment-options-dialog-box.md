# <a name="options-dialog-box-environment"></a>Caixa de diálogo Opções: Ambiente

Você pode usar as páginas da pasta **Ambiente** na caixa de diálogo **Opções** para definir como determinados elementos do IDE (ambiente de desenvolvimento integrado) são exibidos e se comportam. Para acessar as páginas de Ambiente, escolha **Ferramentas** > **Opções** e selecione **Ambiente**.
