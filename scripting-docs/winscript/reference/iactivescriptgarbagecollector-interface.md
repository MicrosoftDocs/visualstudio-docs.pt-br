# <a name="iactivescriptgarbagecollector-interface"></a>Interface IActiveScriptGarbageCollector
Fornece um método para iniciar a coleta de lixo. Essa interface deve ser implementada por mecanismos de Script ativo que deseja limpar seus recursos.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Inicia a coleta de lixo.|
