# <a name="stepunit"></a>STEPUNIT
Especifica a unidade de passo para o passo.

## <a name="syntax"></a>Sintaxe

```cpp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
typedef DWORD STEPUNIT;
```

```csharp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
```

## <a name="fields"></a>Campos
 `STEP_STATEMENT`\
 Passos por declaração.

 `STEP_LINE`\
 Passo a passo.

 `STEP_INSTRUCTION`\
 Passos por instrução.

## <a name="remarks"></a>Comentários
 Passou como um argumento para o método [Step.](../../../extensibility/debugger/reference/idebugprocess3-step.md)
