# <a name="ca0055"></a>CA0055
Não foi possível carregar o arquivo:\<caminho >

 A ferramenta de análise de código não pôde carregar o arquivo de destino foi especificado para análise. Isso ocorre se o arquivo não for encontrado ou o acesso ao arquivo foi negado. Normalmente, isso ocorre se o usuário optou por substituir a propriedade do assembly de entrada no Visual Studio ou chamado fxcopcmd.exe manualmente.
