# <a name="themes-workflow-designer-options-dialog-box-legacy"></a>Temas, Designer de Fluxo de Trabalho, opções da caixa de diálogo (legados)
Este tópico descreve como usar a caixa de diálogo **temas, designer de fluxo de trabalho, opções** no [!INCLUDE[wfd1](../includes/wfd1-md.md)] herdado. Use [!INCLUDE[wfd2](../includes/wfd2-md.md)] herdado quando você precisa definir como alvo [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] ou [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)].

 Um tema define o plano de fundo e de primeiro plano as cores, estilos, ícones, e outros elementos visuais de um fluxo de trabalho. Você pode salvar temas para reutilização por outros fluxos de trabalho. Você pode selecionar temas personalizados padrão e salvos usando a caixa de diálogo **temas, designer de fluxo de trabalho, opções** . Para abrir essa caixa de diálogo, clique em **Selecionar tema personalizado** no menu **fluxo de trabalho** . Ou clique com o botão direito do mouse na superfície de design do fluxo de trabalho e clique em **Selecionar tema personalizado**.

 A tabela a seguir descreve os elementos da interface do usuário da caixa de diálogo **temas, designer de fluxo de trabalho, opções** .

|Elemento da Interface do Usuário|Descrição|
|----------------|-----------------|
|**Novo**|Clique para abrir a [caixa de diálogo configuração de tema (herdada)](../workflow-designer/theme-configuration-dialog-box-legacy.md) para criar um novo tema.|
|**Modificar...**|Clique para abrir a [caixa de diálogo configuração de tema (herdada)](../workflow-designer/theme-configuration-dialog-box-legacy.md) para modificar o tema selecionado.|
|**Procurar**|Clique para procurar um arquivo de tema de fluxo de trabalho, que usa uma extensão de nome de arquivo .wtm.|
|**Selecionar um tema**|Título de coluna do nome do tema de lista dos temas. Selecione um tema na lista, clique em **Modificar** para editar ou clique em **OK** para usar o tema selecionado para o fluxo de trabalho.|
|**Descrição**|Título de coluna de descrição de tema de lista dos temas.|
|**Local**|Título de coluna do local de tema de lista dos temas.|
