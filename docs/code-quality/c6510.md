# <a name="c6510"></a>C6510
C6510 de aviso: Anotação inválida: A propriedade ' NullTerminated ' só pode ser usada em buffers cujos elementos são do tipo integral ou de ponteiro: A função ' \<function > ' \<parameter >.

 Esse aviso indica um uso incorreto da propriedade **NullTerminated** (aquelas que terminam em ' `_z` '). Você só pode usar esse tipo de propriedade em tipos de ponteiro ou matriz.

## <a name="example"></a>Exemplo
 O código a seguir gera este aviso:

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 Para corrigir esse aviso, use o seguinte código:

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
