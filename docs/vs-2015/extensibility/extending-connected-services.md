# <a name="extending-connected-services"></a>Estender serviços conectados
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

O SDK dos serviços conectados do Visual Studio permite que os provedores de serviços criem extensões do Visual Studio que os desenvolvedores de aplicativos podem adicionar facilmente ao seu projeto sem sair do IDE do Visual Studio.
