# <a name="c26498-useconstexprforfunctioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

Essa chamada de função % % de função pode usar `constexpr` se a avaliação do tempo de compilação for desejada. Ver [con.5 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr).
