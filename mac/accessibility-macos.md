# <a name="accessibility-features-of-macos"></a>Recursos de acessibilidade do macOS

O macOS é um sistema operacional acessível, com uma série de recursos para ajudar os usuários de habilidades variadas. Esses recursos incluem um modo de alto contraste, navegação no teclado e VoiceOver (o leitor de tela macOS).

## <a name="enable-accessibility-features"></a>Habilite recursos de acessibilidade

No Visual Studio para Mac, o suporte para tecnologias assistivas é desligado por padrão. Para habilitar o suporte à acessibilidade:

1. Vá para **visual studio (menu)** > **Preferências** > **Outras**e selecione **Acessibilidade**.

1. Selecione a caixa de seleção **Ativar acessibilidade.**

   ![Captura de tela das preferências de acessibilidade, com habilitar acessibilidade selecionada](media/accessibility-preferences.png)

1. Selecione **Reiniciar o Visual Studio** para habilitar o suporte às tecnologias assistivas da Apple.

## <a name="increase-the-contrast-in-macos"></a>Aumente o contraste no macOS

Visual Studio para Mac suporta aumento do contraste no macOS, aumentando o contraste de elementos de IU e tornando os contornos mais definidos. Para habilitar isso:

1. Abrir **preferências do sistema**.

1. Vá para **Acessibilidade**e selecione **Exibir**.

1. Selecione a caixa **de seleção Aumentar contraste.**
