# <a name="spy-menu-commands"></a>Comandos de menu do Spy
O **Spy** menu fornece comandos para trabalhar com Spy + + Windows e modos de exibição.

 **Windows** abre uma nova exibição do Windows.

 **Processos** abre uma nova exibição de processos.

 **Threads** abre uma nova exibição de Threads.

 **As mensagens de log** abre uma nova exibição de mensagens e abre o [caixa de diálogo de opções de mensagem](../debugger/message-options-dialog-box.md) para o modo de exibição.

 **Janela localizar** exibe as **FindWindow** caixa de diálogo para selecionar uma janela com o mouse.

 **Saída** fecha o utilitário Spy + +.
