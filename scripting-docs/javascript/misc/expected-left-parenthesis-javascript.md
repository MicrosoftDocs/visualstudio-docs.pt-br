# <a name="expected--javascript"></a>')' esperado (JavaScript)
Você tentou colocar uma expressão dentro de um conjunto de parênteses, mas não incluiu o parêntese de abertura. Algumas expressões devem ser colocados dentro de um conjunto de abertura e fechamento de parênteses. Observe o uso de parênteses no exemplo a seguir.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o parêntese à esquerda para a expressão de avaliação.
