# <a name="enabling-the-azure-sdk"></a>Habilitando o SDK do Azure
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

SDK do Azure para .NET ajuda a criar, configurar, compilar, depurar, executar e empacotar serviços no Azure e aplicativos Web escalonáveis.  
  
 Você pode baixar o SDK do Azure com o **novo projeto** caixa de diálogo.  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Para habilitar o SDK do Azure  
  
1.  Sobre o **arquivo** menu, crie um novo projeto. (Clique em **New** e, em seguida, clique em **projeto**.)  
  
2.  Expanda o nó para um **Visual Basic** ou **c#** e, em seguida, clique em **nuvem**.  
  
3.  No painel central, selecione **Obtenha o SDK do Azure para .NET**e, em seguida, clique em **Okey**.  
  
     Siga as instruções no instalador.  
  
    > [!NOTE]
    >  Para instalar as ferramentas do Azure, você deve ter o Internet Information Services 7.0 e instalar os componentes de desenvolvimento de aplicativos ASP.NET. Se esses componentes não estiverem instalados, o programa de instalação de ferramentas do Azure solicitará que você instalá-los.
