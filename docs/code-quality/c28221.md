# <a name="c28221"></a>C28221
aviso C28221: expressão de cadeia de caracteres esperada para o parâmetro na anotação

 Esse aviso indica que um parâmetro para uma anotação deve ser uma cadeia de caracteres e algum outro tipo foi encontrado. Isso geralmente indica uma macro de anotação codificada incorretamente e não deve ser visto no uso normal.
