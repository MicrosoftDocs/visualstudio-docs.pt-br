# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>Não foi possível recuperar informações de esquema para o nome do objeto de objeto de banco de dados &lt;&gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Essa mensagem normalmente aparece quando um objeto em **Gerenciador de Servidores/Gerenciador de banco de dados** é copiado para a área de transferência, excluído do banco de dados e, em seguida, colado no designer. Porque o objeto de base de dados não existir, esta mensagem aparece.

### <a name="to-correct-this-error"></a>Para corrigir este erro

- Clique **OK** para descartar a caixa de diálogo.
