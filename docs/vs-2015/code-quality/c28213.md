# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28213: a `_Use_decl_annotations_` anotação deve ser usada para fazer referência, sem modificação, uma declaração anterior. \<why>  
  
 Esse aviso é relatado quando uma declaração anterior é referenciada sem a `_Use_decl_annotations_` anotação necessária.
