# <a name="warning-c26453-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>C26453 de aviso: Estouro aritmético: O deslocamento à esquerda de um número negativo assinado é um comportamento indefinido

Esse aviso indica que deixamos de lado um valor integral assinado negativo, que é uma má ideia e dispara um comportamento definido pela implementação.

## <a name="example"></a>Exemplo

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

Para corrigir esse aviso, use o seguinte código:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
