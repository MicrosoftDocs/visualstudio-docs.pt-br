# <a name="c6518"></a>C6518
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C6518: anotação inválida: a propriedade ' SAL_writableTo ' não pode ser especificada como uma pré-condição em buffers que não são graváveis:*' \\ param*(1) '  
  
 Esse aviso indica que existe um conflito entre um `SAL_writableTo` valor de propriedade e uma propriedade gravável. Isso normalmente indica que uma propriedade gravável não tem acesso de gravação ao parâmetro que está sendo anotado.  
  
## <a name="example"></a>Exemplo  
 O código a seguir gera esse aviso porque a `_Out_` anotação é compilada para incluir uma `SAL_writableTo` propriedade, que não permite acesso de gravação:  
  
```cpp  
#include <sal.h>  
void f(_Out_ const char* pc)  
{  
    //code that can't write to *pc ...  
}  
```  
  
 Para corrigir esse aviso, use o seguinte código:  
  
```cpp  
#include <sal.h>  
void f(_Out_  char* pc)  
{  
    pc = "Hello World";  
    //code ...  
}  
```
