## <a name="im-unable-to-download-or-activate-software-excluding-visual-studio-family"></a>Não consigo baixar ou ativar o software (excluindo a família Visual Studio)

Os downloads de software disponíveis dependem da assinatura do Visual Studio que você está usando. Você pode encontrar downloads e chaves de produto de algumas maneiras.  

## <a name="excel-list-of-available-software"></a>Lista de softwares disponíveis do Excel 
Você pode [baixar uma lista extensa](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)   de títulos de software disponíveis para todos os tipos de assinatura. Para verificar quais tipos estão atribuídos a você, exiba suas assinaturas no [portal de assinaturas](https://my.visualstudio.com/subscriptions).  

## <a name="find-software-downloads"></a>Encontre downloads de software 
1. Entre no [portal de assinaturas](https://my.visualstudio.com/benefits).  
1. Vá para a [página de downloads em destaque](https://my.visualstudio.com/downloads/featured).  
1. Aqui você poderá ver uma lista de downloads populares ou pesquisar um produto específico. Observação: se você tiver mais de uma assinatura associada ao mesmo endereço de entrada, você verá os downloads incluídos com a assinatura de nível mais alto, independentemente da assinatura que você está exibindo.  
4. Clique em "obter chave" para ver todas as chaves que estão disponíveis para o produto que você deseja baixar. Em seguida, clique em "chave de declaração" para reivindicar uma chave do produto se o download exigir uma chave para começar. 

## <a name="product-keys-page"></a>Página chaves do produto 
Você também pode visitar a página de [chaves do produto](https://my.visualstudio.com/productkeys),   onde todas as chaves estão disponíveis na agregação. Você pode editar a coluna observações para identificar quando e por que a chave foi solicitada. 

## <a name="more-information"></a>Mais informações 
[Leia mais](https://docs.microsoft.com/visualstudio/subscriptions/find-keys) sobre como localizar e reivindicar downloads e chaves de produto.
