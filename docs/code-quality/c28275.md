# <a name="c28275"></a>C28275
C28275 de aviso: O parâmetro para \_Macro @ no__t-1value @ no__t-2 é nulo

 Esse aviso indica que há um erro interno no arquivo de modelo, não no código que está sendo analisado. A função *macrovalue* foi chamada sem um parâmetro.
