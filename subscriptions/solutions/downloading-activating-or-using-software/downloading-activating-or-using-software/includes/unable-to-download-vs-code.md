## <a name="im-unable-to-download-or-activate-visual-studio-code"></a>Não consigo baixar ou ativar Visual Studio Code

Visual Studio Code é um editor de código gratuito e você não precisa de uma assinatura paga do Visual Studio para usá-lo. Você pode [baixá-lo diretamente](https://code.visualstudio.com/download) do site do Visual Studio.
