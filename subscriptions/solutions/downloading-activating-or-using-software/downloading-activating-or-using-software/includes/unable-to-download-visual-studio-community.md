## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>Não consigo baixar ou ativar a Comunidade do Visual Studio
A Comunidade do Visual Studio é a versão leve e gratuita do IDE e você não precisa de uma assinatura paga do Visual Studio para usá-la. Você pode [baixá-lo diretamente](https://visualstudio.microsoft.com/vs/community/) do site do Visual Studio.
