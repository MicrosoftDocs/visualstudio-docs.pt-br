# <a name="warning-c26812-prefer-enum-class-over-enum-enum3"></a>Aviso C26812 prefira ' enum Class ' sobre ' Enum ' (Enum. 3)
O tipo de enumeração% Type% não tem escopo. Prefira ' enum Class ' sobre ' Enum ' (Enum. 3)

## <a name="example"></a>Exemplo
O exemplo a seguir é das C++ principais diretrizes

```cpp
void Print_color(int color);

enum Web_color { red = 0xFF0000, green = 0x00FF00, blue = 0x0000FF }; // C26812 reported here
enum Product_info { Red = 0, Purple = 1, Blue = 2 }; // C26812 reported here

Web_color webby = Web_color::blue;

// Clearly at least one of these calls is buggy.
Print_color(webby);
Print_color(Product_info::Blue);
```

Esses avisos são corrigidos declarando o enum como classe enum:

```cpp
void Print_color(int color);

enum class Web_color { red = 0xFF0000, green = 0x00FF00, blue = 0x0000FF }; // no C26812
enum class Product_info { red = 0, purple = 1, blue = 2 }; // no C26812

Web_color webby = Web_color::blue;
Print_color(webby);  // Error: cannot convert Web_color to int.
Print_color(Product_info::Red);  // Error: cannot convert Product_info to int.
```
