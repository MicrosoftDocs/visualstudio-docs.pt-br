# <a name="c6518"></a>C6518
C6518 de aviso: Anotação inválida: A propriedade ' SAL_writableTo ' não pode ser especificada como uma pré-condição em buffers que não são graváveis: ' \_Param @ no__t-1 (1) '

 Esse aviso indica que existe um conflito entre um valor de propriedade `SAL_writableTo` e uma propriedade gravável. Isso normalmente indica que uma propriedade gravável não tem acesso de gravação ao parâmetro que está sendo anotado.

## <a name="example"></a>Exemplo
 O código a seguir gera esse aviso porque a anotação `_Out_` é compilada para incluir uma propriedade `SAL_writableTo`, que não permite acesso de gravação:

```cpp
#include <sal.h>
void f(_Out_ const char* pc)
{
    //code that can't write to *pc ...
}
```

 Para corrigir esse aviso, use o seguinte código:

```cpp
#include <sal.h>
void f(_Out_  char* pc)
{
    pc = "Hello World";
    //code ...
}
```
