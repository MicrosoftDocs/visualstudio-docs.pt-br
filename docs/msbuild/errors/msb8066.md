# <a name="msb8066--custom-build-for-item-list-exited-with-code-error-code"></a>MSB8066: a compilação personalizada para ' item-List ' saiu com o código ' código de erro '

Esse erro ocorre quando uma compilação personalizada especificada para uma lista de itens foi encerrada com um código de erro diferente de zero.

Para investigar o erro, verifique a saída da compilação ou o log de compilação para ver se a etapa de compilação personalizada produziu um erro com informações sobre o problema. Caso contrário, tente executá-lo separadamente para verificar se há problemas ou comportamento inesperado.

Consulte [noções básicas sobre etapas de compilação personalizadas e eventos de compilação](/cpp/build/understanding-custom-build-steps-and-build-events)
