# <a name="c28279"></a>C28279
aviso C28279: for Symbol, um ' Begin ' foi encontrado sem um ' End ' correspondente

 A linguagem de anotação dá suporte a uma construção de início e término (`{` e `}` em C) e o emparelhamento ficou desbalanceado. Essa situação pode ser evitada se as macros forem usadas.
