# <a name="unterminated-string-constant-javascript"></a>Constante de cadeia de caracteres não finalizada (JavaScript)
Você não finalizou sua constante de cadeia de caracteres com uma aspa de fechamento. As constantes de cadeia de caracteres devem ser colocadas dentro de um par de aspas.  
  
> [!NOTE]
> Você pode usar pares correspondentes de aspas simples ou duplas. Aspas duplas podem ser contidas em cadeias de caracteres entre aspas simples, e aspas simples podem ser contidas em cadeias de caracteres entre aspas duplas.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione a aspa de fechamento ao final da cadeia de caracteres.
