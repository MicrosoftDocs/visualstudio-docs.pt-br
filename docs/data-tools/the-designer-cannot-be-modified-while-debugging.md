# <a name="the-designer-cannot-be-modified-while-debugging"></a>O designer não pode ser alterado para depuração

Esta mensagem aparece quando é feita uma tentativa de modificar itens sobre os **Relational Designer** quando o aplicativo está em execução no modo de depuração. Quando o aplicativo está em execução no modo de depuração, o **Relational Designer** é somente leitura.

Para corrigir esse erro, selecione **parar depuração** sobre o **depurar** menu. O aplicativo para depuração, e você pode modificar itens de **Relational Designer**.
