# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset
Redefine uma sequência de enumeração para o início.  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Retorna `S_OK`.
