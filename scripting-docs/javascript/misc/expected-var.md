# <a name="expected-var"></a>'var' esperado
Esperado ' var ' (ERRnoVar 1034))  
  
 Falha ao adicionar a palavra-chave var quando ela era necessária.  
  
 Para obter mais informações sobre a palavra-chave var, consulte [var Statement](../../javascript/reference/var-statement-javascript.md).
