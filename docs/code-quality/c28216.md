# <a name="c28216"></a>C28216
aviso C28216: o \_verificar\_\_ anotação de retorno se aplica somente a pós-condições para o parâmetro de função específico.

 A anotação `_Check_return_` foi aplicada em um contexto diferente de `__post`; Ele pode precisar de um modificador `__post` (ou `__drv_out`) ou pode ser colocado incorretamente.
