# <a name="objects"></a>Objetos
As seguintes informações de referência explicam objetos que são usados para criar uma instância do editor de texto principal.

## <a name="in-this-section"></a>Nesta seção
- [Objeto VSCodeWindow](../extensibility/vscodewindow-object.md) Descreve o objeto da janela de código, uma janela de documento especializada que pode envolver uma ou mais exibições de texto.

- [Objeto VSCodeWindowManager](../extensibility/vscodewindowmanager-object.md) Descreve o objeto gerenciador de janela de código que é implementado pelo serviço de idiomas e é responsável pelo gerenciamento de adornos.

- [Objeto VSTextBuffer](../extensibility/vstextbuffer-object.md) Descreve o objeto buffer de texto, que representa um fluxo de texto Unicode, os dados de um arquivo.

- [Objeto VSTextView](../extensibility/vstextview-object.md) Descreve a exibição de texto, uma janela que permite aos usuários visualizar e editar texto Unicode do buffer de texto.
