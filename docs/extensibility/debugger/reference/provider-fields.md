# <a name="provider_fields"></a>PROVIDER_FIELDS
Especifica propriedades associadas a um provedor de programa.

## <a name="syntax"></a>Sintaxe

```cpp
enum enum_PROVIDER_FIELDS {
   PFIELD_PROGRAM_NODES       = 0x01,
   PFIELD_IS_DEBUGGER_PRESENT = 0x02
};
typedef DWORD PROVIDER_FIELDS;
```

```csharp
public enum enum_PROVIDER_FIELDS {
   PFIELD_PROGRAM_NODES       = 0x01,
   PFIELD_IS_DEBUGGER_PRESENT = 0x02
};
```

## <a name="fields"></a>Campos
 `PFIELD_PROGRAM_NODES`\
 O `ProgramNodes` campo é válido.

 `PFIELD_IS_DEBUGGER_PRESENT`\
 O `fIsDebuggerPresent` campo é válido.

## <a name="remarks"></a>Comentários
 Esses valores são `Fields` devolvidos no membro da estrutura [PROVIDER_PROCESS_DATA](../../../extensibility/debugger/reference/provider-process-data.md) para indicar quais campos da estrutura foram explicitamente preenchidos.

 Esses valores podem ser combinados com um pouco `OR`.
