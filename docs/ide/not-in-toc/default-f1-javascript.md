# <a name="javascript-f1-help-for-visual-studio"></a>Ajuda F1 do JavaScript para Visual Studio

O JavaScript é uma linguagem de script que pode ser inserida em páginas da Web e em outros aplicativos. Para o Microsoft Edge herdado e o Internet Explorer, a implementação do JavaScript da Microsoft é compatível com a edição da linguagem ECMAScript 5º. Ela também fornece recursos adicionais que não fazem parte dos padrões ECMA.

Para obter ajuda de referência sobre palavras-chave de idioma e elementos de código, consulte:

- [MDN Web docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) para obter ajuda com o JavaScript. Na caixa de pesquisa, digite o nome da palavra-chave que você está procurando.
- [Documentação do typescript](https://www.typescriptlang.org/docs) para obter ajuda com o typescript.
- [Página de pesquisa](/search) em docs.Microsoft.com se você estiver procurando ajuda na documentação oficial da Microsoft.

> [!IMPORTANT]
> A Microsoft faz parte do esforço de toda a Comunidade para tornar o [MDN Web docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) a referência de desenvolvimento One-Stop, Premiere da Web. Todas as mais de 500 páginas da referência de API JavaScript da Microsoft foram modeladas de [docs.Microsoft.com](https://docs.microsoft.com) para suas contrapartes MDN. Nossa referência de extensões JS herdadas também foram portadas para fins de serviço e históricos.
