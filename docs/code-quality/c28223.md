# <a name="c28223"></a>C28223
Warning C28223: O token/identificador esperado para anotação não encontrado, parâmetro

 Este aviso indica que um parâmetro em uma anotação deve ser um identificador, e alguns outros símbolos foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente. O uso de palavras-chave C ou C++ nesta posição fará com que esse erro.
