# <a name="color-value-reference-for-visual-studio"></a>Referência de valor de cor para Visual Studio

Este tópico é uma referência para a criação de novos tokens de cores. Ao criar uma nova interface do usuário, você pode usar cores semelhantes da interface do usuário existente para escolher as cores de cada tema principal e o nome da cor do sistema para esse elemento Alto Contraste temas.

Para obter informações de implementação sobre como criar novos tokens de cores, consulte [Ferramentas de temas de cores](../../extensibility/internals/color-theming-tools.md).

Para tokens de cores a usar com a interface do usuário existente, consulte [Cores compartilhadas para Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

Ao usar este guia de referência, lembre-se:

- **Sempre copie os matizes de primeiro plano e de plano de fundo juntos.** Não escolha um primeiro plano de um elemento de interface do usuário e uma plano de fundo de outro apenas porque você gosta dos matizes.

- **Esses são códigos hexatórios, não tokens de cor.** Você não pode usar os nomes reais de token de cor desses elementos da interface do usuário porque não é possível controlar como os tokens podem ser alterados no futuro. Em vez disso, crie tokens em sua própria categoria e use os valores de matiz mostrados nas tabelas. Os tokens de cor para a interface do usuário existente podem ser encontrados em [Cores compartilhadas para Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

- **Alto Contraste são mostradas apenas para comparação.** Use o nome da cor do sistema para Alto Contraste interface do usuário e não use um seta-gotas ou selador de cores nessas travas. Alto Contraste neste tópico refletem o Alto Contraste #1 no Windows. As cores serão diferentes em outros Alto Contraste temas.

- **Para melhores resultados, emparelhe conjuntos de cores com interface do usuário semelhante ao exemplo.**
