# <a name="create-a-wpf-based-domain-specific-language"></a>Criar uma Linguagem Específica de Domínio baseada no WPF

Você pode criar uma linguagem específica de domínio que tenha um designer WPF em vez de um designer gráfico.

Informações e exemplos sobre esse recurso podem ser encontrados no site de ferramentas de visualização e modelagem do Visual Studio em [http://go.microsoft.com/fwlink/?LinkId=186128](https://code.msdn.microsoft.com/Visualization-and-Modeling-313535db)
