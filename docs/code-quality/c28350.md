# <a name="c28350"></a>C28350
C28350 de aviso: A anotação \<annotation > descreve uma situação que não é condicionalmente aplicável.

 Normalmente, esse aviso é gerado quando uma anotação é aplicada onde o CC++ /Type está sendo inspecionado.
