# <a name="c28218"></a>C28218
Aviso C28218: para parâmetro de função, o parâmetro de anotação não corresponde ao encontrado no arquivo

 As anotações na linha atual e na linha da mensagem são inconsistentes. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
