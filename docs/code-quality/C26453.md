# <a name="warning-c26253-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>Aviso C26253: Estouro aritmético: Deslocamento à esquerda de um número negativo assinado é um comportamento indefinido

Este aviso indica que são deixamos o deslocamento de um negativo valor de integral com sinal, que é uma má ideia e dispara o comportamento definido pela implementação.

## <a name="example"></a>Exemplo

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

 Para corrigir este aviso, use o seguinte código:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
