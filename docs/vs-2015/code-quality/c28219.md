# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28219: membro de enumeração esperado para a anotação do parâmetro na anotação  
  
 Um parâmetro para uma anotação deve ser um membro do `enum` tipo nomeado e algum outro símbolo foi encontrado; use um membro desse `enum` tipo. Isso geralmente indica uma macro de anotação codificada incorretamente.
