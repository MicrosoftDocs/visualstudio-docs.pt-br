> [!NOTE]
> O teste de IU codificado para teste funcional controlado por interface do usuário automatizado foi preterido. O Visual Studio 2019 será a última versão em que o teste de IU codificado estará disponível. É recomendável usar o [Selenium](https://docs.seleniumhq.org/) para testar aplicativos Web e [Appium com WinAppDriver](https://github.com/Microsoft/WinAppDriver) para testar aplicativos de área de trabalho e da UWP. Considere [Xamarin.UITest](/appcenter/test-cloud/uitest/) para testar aplicativos iOS e Android usando a estrutura de teste NUnit.
