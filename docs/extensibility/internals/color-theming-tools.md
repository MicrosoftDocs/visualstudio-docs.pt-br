# <a name="color-theme-tools"></a>Ferramentas temáticas de cores
O Visual Studio SDK fornece várias ferramentas para ajudá-lo a criar e gerenciar temas de cores.

- A ferramenta [VSIX Color Editor](../../extensibility/internals/vsix-color-editor.md) pode criar e editar cores personalizadas para o Visual Studio.

- A ferramenta [VSIX Color Compiler](../../extensibility/internals/vsix-color-compiler.md) converte arquivos do Visual Studio *.xml* existentes em arquivos *.pkgdef* para sua extensão.
