## <a name="what-is-the-difference-between-using-a-microsoft-account-vs-work-account"></a>Qual é a diferença entre o uso de uma conta Microsoft vs. uma conta corporativa?

Uma MSA (conta Microsoft) é uma conta de email pessoal de propriedade de um indivíduo usada para acessar os serviços da Microsoft quando uma organização não está usando um locatário gerenciado do Office 365 ou do Azure Active Directory. Uma conta corporativa é uma conta que é de propriedade da organização que a emite e é usada para acessar os serviços nos locatários do Office 365 ou do Azure Active Directory. Para obter informações completas sobre as diferenças entre MSAs e contas corporativas, confira ['Saiba mais sobre as diferenças entre o uso de MSAs (contas Microsoft) vs. contas corporativas (Azure Active Directory ou Office 365) e como isso pode afetar a sua organização'](https://aka.ms/MSAvsAAD).
