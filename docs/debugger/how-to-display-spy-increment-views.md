# <a name="how-to-display-spy-views"></a>Como mostrar exibições do Spy++
Quando Spy + + é iniciado, ele é aberto em [exibição Windows](../debugger/windows-view.md), uma exibição de árvore de todas as janelas e os controles disponíveis em seu computador. Esta janela de raiz é legenda "Windows 1". Também há três outros Spy + + modos de exibição: [exibição de mensagens](../debugger/messages-view.md), [exibição processos](../debugger/processes-view.md), e [exibição de Threads](../debugger/threads-view.md). Você pode exibir o modo de exibição desejado do **Spy** menu. Para obter mais informações, consulte [exibições do Spy + +](../debugger/spy-increment-views.md).
