# <a name="msb3103-invalid-resx-file"></a>MSB3103: Arquivo Resx inválido

Esse erro ocorre quando há um problema com um arquivo de recurso .resx. Quando você vir esse erro, outro erro será exibido com mais detalhes. Procure o outro erro para obter mais informações.
