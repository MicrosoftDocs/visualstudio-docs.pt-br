# <a name="microsoft-qa-forum-support-in-visual-studio-subscriptions"></a>Microsoft Q&um suporte A Fórum em assinaturas do Visual Studio
Tem uma pergunta técnica sobre o produto? Encontre-o no Microsoft Q&a – a página inicial para perguntas e respostas técnicas na Microsoft.

Você pode fazer perguntas e respostas em uma ampla variedade de produtos com suporte, incluindo:
- .NET
- Azure
- C++
- Enterprise Mobility + Security
- Microsoft 365
- Microsoft Advertising
- Microsoft Edge
- Microsoft Robotics
- Open Specifications
- Power Query 
- Básico pequeno
- SQL Server
- System Center
- Visual Studio
- Windows 
- Windows Sysinternals

Você também encontrará guias "como fazer" para ajudá-lo a fazer perguntas incríveis e obter respostas de qualidade para outros usuários. 

## <a name="access-microsoft-qa"></a>Acessar o Microsoft Q&A
O acesso ao Microsoft Q&A é simples. 
1. Entre no portal de assinaturas do Visual Studio em <https://my.visualstudio.com/benefits>
0. Localize o Microsoft Q&um bloco na seção de suporte e clique em **obter ajuda**.

   > [!div class="mx-imgBorder"]
   > ![Microsoft Q&um bloco](_img/vs-microsoft-qa/vs-microsoft-qa-tile.png "Clique em ' obter ajuda ' para se conectar ao Microsoft Q&um fórum")

## <a name="eligibility"></a>Qualificação
A disponibilidade do benefício Comunidade Azure depende do nível de sua assinatura, conforme mostrado abaixo:

|                                          Nível de Assinatura                                           |     Canais      |    Benefício    | Renovável? |
|-------------------------------------------------------------------------------------------------------|-------------------|---------------|------------|
|                           Visual Studio Enterprise (padrão), NFR<sup>1</sup>                            | VL, Azure, Retail |   Disponível    |    Sim     |
|                           Assinatura Visual Studio Enterprise com o GitHub Enterprise                           | VL, Azure, Retail |   Disponível    |    Sim     |
|                          Visual Studio Professional (Standard)                          | VL, Azure, Retail |   Disponível    |    Sim     |
|                          Assinatura Visual Studio Professional com o GitHub Enterprise                          | VL, Azure, Retail |   Disponível    |    Sim     |
|                              Visual Studio Test Professional (Padrão)                               |    VL, Retail     |   Disponível    |    Sim     |
|                                       Plataformas MSDN (Padrão)                                       |    VL, Retail     |   Disponível    |    Sim     |
| Visual Studio Enterprise (nuvem mensal), Visual Studio Professional (nuvem mensal)|        Tudo        | Não disponível |     Não     |

<sup>1</sup>  *inclui: não para revenda (NFR), Microsoft FTE, profissional mais valioso (MVP), diretor regional (RD), parceiro do setor do Visual Studio (VSIP), BizSpark, imagine, Microsoft Certified Trainer (MCT)*

> [!NOTE]
> A Microsoft não oferece mais assinaturas anuais do Visual Studio Professional e do Visual Studio Enterprise nas Assinaturas na Nuvem. Não haverá nenhuma alteração na experiência dos clientes existentes nem na capacidade de renovar, aumentar, diminuir ou cancelar suas assinaturas. Novos clientes são incentivados a ir para [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) a fim de explorar diferentes opções para comprar o Visual Studio.

Não tem certeza de qual assinatura você está usando?  Conecte-se ao [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) para ver todas as assinaturas atribuídas ao seu endereço de email. Se não vir todas as suas assinaturas, talvez você tenha uma ou mais atribuídas a outro endereço de email.  Você precisará entrar com esse endereço de email para ver as assinaturas.
