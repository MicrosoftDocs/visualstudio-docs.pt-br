# <a name="ca0503"></a>CA0503
A propriedade foi preterida. Usar a propriedade substituta

Aviso CA0503 é gerado quando uma propriedade de análise de código preterida contém um valor no arquivo de configuração de projeto ou compilação. Para resolver esse aviso, use a propriedade especificada que o substitui.
