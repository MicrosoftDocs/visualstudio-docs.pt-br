# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28229: A anotação não tem suporte a parâmetros  
  
 Este aviso indica que uma anotação foi usada com um parâmetro quando a anotação é declarada sem parâmetros. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso típico.
