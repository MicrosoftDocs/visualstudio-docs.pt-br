# <a name="c28232"></a>C28232
Aviso C28232: \_Pré\_, \_Post\_, ou \_Deref\_ não aplicado a qualquer anotação

 Este aviso indica que um `_Pre_`, `_Post_`, ou `_Deref_` operador aparece em uma expressão de anotação sem uma anotação funcional subsequente; o modificador foi ignorado, mas isso indica uma anotação codificada incorretamente.
