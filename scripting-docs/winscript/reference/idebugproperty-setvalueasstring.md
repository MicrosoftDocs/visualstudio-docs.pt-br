# <a name="idebugpropertysetvalueasstring"></a>IDebugProperty::SetValueAsString
Define o valor de uma propriedade de uma determinada cadeia de caracteres.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT SetValueAsString (  
   LPCOLESTR pszValue,  
   UINTnRadix,  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pszValue`  
 no O valor a ser definido.  
  
 `nRadix`  
 no Base a ser usada na interpretação de qualquer informação numérica.  
  
## <a name="return-value"></a>Valor retornado  
 Retorna um `HRESULT` válido, geralmente `S_OK`.
