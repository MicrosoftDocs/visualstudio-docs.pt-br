# <a name="c6530"></a>C6530
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso 6530: estilo de cadeia de caracteres de formato não reconhecido \<name>  
  
> [!NOTE]
> Esse aviso ocorre apenas no código que está usando uma versão preterida do SAL (código-fonte de anotação). É recomendável que você porte seu código para usar a versão mais recente do SAL. Para obter mais informações, consulte [usando anotações de sal para reduzir os defeitos de código C/C++](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).  
  
 Esse aviso indica que a Propriedade FormatString está usando um valor diferente de scanf ou printf. Para corrigir esse aviso, examine seu código e use um valor válido para a propriedade Style.  
  
## <a name="example"></a>Exemplo  
 O código a seguir gera esse aviso devido a um erro de digitação na `Style` Propriedade:  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printfd")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printfd")] char *px);  
  
```  
  
 Para corrigir esse aviso, use um estilo válido, conforme mostrado no código a seguir:  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printf")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printf")] char *px);   
```
