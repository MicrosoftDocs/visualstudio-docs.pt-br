# <a name="tree-menu-commands"></a>Comandos do menu Árvore
O menu de **árvore** fornece comandos para expandir ou recolher exibições.

 **Expandir um nível** Expande o item atualmente selecionado em uma exibição do sistema exibindo um nível abaixo dele.

 **Expandir ramificação** Expande todos os níveis abaixo do item selecionado no momento.

 **Expandir tudo** Expande e exibe todos os níveis em uma exibição do sistema.

 **Recolher** Recolhe todos os níveis abaixo do item selecionado no momento.
