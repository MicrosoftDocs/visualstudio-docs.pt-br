# <a name="process-properties-dialog-box"></a>Caixa de diálogo Propriedades do Processo
Use essa caixa de diálogo para obter mais informações sobre um processo específico. Para exibir essa caixa de diálogo, mova o foco para um [exibição de processos](../debugger/processes-view.md) janela. Selecione qualquer nó de processo na árvore e escolha **propriedades** da **exibição** menu.

 Em sistemas de operacionais Windows 98 e Windows 95, somente uma versão modificada do **geral** guia é exibida.

## <a name="in-this-section"></a>Nesta seção
 [Guia geral, caixa de diálogo de propriedades do processo](../debugger/general-tab-thread-properties-dialog-box.md) Use essa caixa de diálogo obter mais informações sobre o processo selecionado.

 [Guia memória, caixa de diálogo de propriedades do processo](../debugger/memory-tab-process-properties-dialog-box.md) Use esta guia para mostrar como um processo usa memória.

 [Guia arquivo de paginação, caixa de diálogo de propriedades do processo](../debugger/page-file-tab-process-properties-dialog-box.md) Use esta guia para examinar o arquivo de paginação de um processo.

 [Guia espaço, caixa de diálogo de propriedades do processo](../debugger/space-tab-process-properties-dialog-box.md) Use esta guia para examinar o espaço de endereço de um processo.
