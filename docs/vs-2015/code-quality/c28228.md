# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28228: parâmetro de anotação: não é possível usar tipo em anotações  
  
 Este aviso indica que um parâmetro é do tipo que não tem suporte. Anotações só podem usar um conjunto limitado de tipos como parâmetros. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso típico.
