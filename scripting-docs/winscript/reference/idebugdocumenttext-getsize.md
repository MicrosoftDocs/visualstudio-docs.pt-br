# <a name="idebugdocumenttextgetsize"></a>IDebugDocumentText::GetSize
Retorna o número de linhas e o número de caracteres no documento.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT GetSize(  
   ULONG*  pcNumLines,  
   ULONG*  pcNumChars  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pcNumLines`  
 fora Número de linhas no documento. Se esse parâmetro for nulo, o método não retornará um valor.  
  
 `pcNumChars`  
 fora Número de caracteres no documento. Se esse parâmetro for nulo, o método não retornará um valor.  
  
## <a name="return-value"></a>Valor retornado  
 O método retorna um `HRESULT`. Os possíveis valores incluem, mas sem limitação, aqueles na tabela a seguir.  
  
|Valor|Descrição|  
|-----------|-----------------|  
|`S_OK`|O método foi bem-sucedido.|  
  
## <a name="remarks"></a>Comentários  
 Esse método retorna o número de linhas e o número de caracteres no documento.
