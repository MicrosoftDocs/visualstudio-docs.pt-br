# <a name="c28288"></a>C28288
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28288: para a função, erro de sintaxe na \_ \_ anotação at () (nome de parâmetro inválido)  
  
 A ferramenta de análise de código relata esse aviso quando a `SAL_at` `__drv_at` Anotação () é usada e a expressão de parâmetro não pode ser interpretada no contexto atual. Isso pode incluir o uso de um parâmetro ou nome de membro digitado incorretamente ou uma grafia incorreta das palavras-chave "Return" ou "This".
