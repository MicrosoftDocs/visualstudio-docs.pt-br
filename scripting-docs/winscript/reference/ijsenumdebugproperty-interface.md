# <a name="ijsenumdebugproperty-interface"></a>Interface IJsEnumDebugProperty
Fornece operações de procura de propriedades.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
IJsEnumDebugProperty : public IUnknown;  
```  
  
## <a name="members"></a>Membros

|Nome|Descrição|  
|----------|-----------------|  
|[Método IJsEnumDebugProperty::GetCount](../../winscript/reference/ijsenumdebugproperty-getcount-method.md)|Obtém o número de propriedades do objeto.|  
|[Método IJsEnumDebugProperty::Next](../../winscript/reference/ijsenumdebugproperty-next-method.md)|Lê propriedades para este objeto.|
