# <a name="keyboard-navigation-in-visual-studio-for-mac"></a>Navegação de teclado no Visual Studio para Mac

Visual Studio para Mac é acessível por usuários de teclado usando atalhos de teclado macOS mais comuns, bem como muitos específicos para Visual Studio para Mac. Aqui, abordaremos alguns comandos básicos de teclado para ambos e demonstraremos como configurar Visual Studio para Mac atalhos de teclado.

## <a name="how-to-use-keyboard-navigation"></a>Como: Usar navegação do teclado

O suporte à navegação por teclado é criado diretamente no macOS, mas para ter a experiência mais abrangente, você deve definir o macOS para navegar por **todos os controles**:

![Todos os controles do teclado de preferências do sistema](media/accessibility-preferences-keyboard.png)

Definir o **Acesso completo por teclado** para **Todos os controles** permite que você navegue por todos os controles em uma janela ou caixa de diálogo.

## <a name="customizing-keyboard-shortcuts"></a>Personalizando atalhos de teclado

Visual Studio para Mac tem uma variedade de atalhos de teclado para várias tarefas. Você pode exibir e editar esses atalhos da seguinte maneira:

1. Abra as preferências de Visual Studio para Mac usando o **Visual Studio (menu) >&#8984;preferências (,)**

1. Navegue até o **ambiente > associações de chave**

1. Percorrer a lista ou pesquisar o atalho ou comando

1. Selecione o comando

1. Selecione **Associação de chave** e digite uma combinação de teclas

1. Selecione **aplicar**

> [!TIP]
> Você pode escolher em uma lista de esquemas de associação de chave para atender melhor às suas necessidades se estiver familiarizado com outros atalhos de teclado comuns, por exemplo, Visual Studio no Windows.

|Consegue        |Selecionar o controle padrão na tela/caixa de diálogo |**⏎**            |
