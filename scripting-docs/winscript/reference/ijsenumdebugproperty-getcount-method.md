# <a name="ijsenumdebugpropertygetcount-method"></a>Método IJsEnumDebugProperty::GetCount
Obtém o número de propriedades do objeto.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pCount`  
 fora O número de propriedades.  
  
## <a name="return-value"></a>Valor retornado  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jscript9diag. h
