# <a name="ca0055"></a>CA0055
Não foi possível carregar o arquivo:\<path>

A ferramenta de análise de código não pôde carregar o arquivo de destino que foi especificado para análise. Isso ocorrerá se o arquivo não for encontrado ou se o acesso ao arquivo tiver sido negado. Normalmente, isso ocorre se o usuário optasse por substituir a propriedade do assembly de entrada no Visual Studio, ou invocado fxcopcmd.exe manualmente.
