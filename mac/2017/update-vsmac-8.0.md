# <a name="update-visual-studio-2017-for-mac-version-7-to-visual-studio-2019-for-mac-version-80"></a>Atualize o Visual Studio 2017 para Mac (versão 7) para o Visual Studio 2019 para Mac (versão 8.0)

O Visual Studio 2019 para Mac traz diversas novas melhorias: um novíssimo editor C#, uma nova experiência de inicialização, suporte para diversas instâncias e vários novos aprimoramentos para desenvolvedores Xamarin, Unity e .NET Core. Para saber mais sobre tudo que está incluso no Visual Studio para Mac 2019, confira as postagens de blog a seguir:

- [O Visual Studio 2019 para Mac já está disponível](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-is-now-available/)
- [O Visual Studio 2019 para Mac versão 8.1 já está disponível (e uma versão prévia para teste da 8.2)](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-version-8-1-is-now-available-and-a-preview-for-8-2/)

> [!IMPORTANT]
> Há um problema conhecido com o Visual Studio 2017 para Mac **versão 7.7.3**, em que o atualizador continuamente mostrará "Erro ao recuperar informações de atualização". Se isso acontecer, [Baixe o Visual Studio para Mac](https://visualstudio.microsoft.com/vs/mac/) e siga as etapas para [instalar](/visualstudio/mac/installation?view=vsmac-2019) o Visual Studio 2019 para Mac. Observe que isso substituirá a instância existente do Visual Studio 2017 para Mac.

Para obter a versão mais recente, você poderá usar o [atualizador](/visualstudio/mac/update?view=vsmac-2017) navegando até **Visual Studio > Verificar se há Atualizações...**:

![caixa de diálogo do atualizador](media/update-vsmac-updater.png)

Ou, se você preferir ou estiver tendo problemas com o atualizador, [baixe](https://visualstudio.microsoft.com/vs/mac/) o novo instalador da versão 2019 e entre em operação rapidamente:

![tela de instalação](media/update-vsmac-installer.png)
