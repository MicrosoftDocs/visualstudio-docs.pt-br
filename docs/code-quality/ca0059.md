# <a name="ca0059"></a>CA0059
Erro de opção de linha de comando.

 Esse erro ocorre se alternar de uma linha de comando necessária está ausente, por exemplo,

 /out ou opção /console não for especificada. Esse erro também ocorrerá se houver uma opção inválida, como um valor de tempo limite inválido.
