## <a name="what-software-downloads-are-available"></a>Quais downloads de software estão disponíveis?

Os downloads de software disponíveis dependem das assinaturas que você tem. Se você tiver mais de uma assinatura, os downloads da assinatura de nível mais alto ficarão visíveis em todas as suas assinaturas. Isso facilita a obtenção dos downloads que você deseja sem alternar entre assinaturas.

Confira esta [lista dos downloads disponíveis por nível de assinatura](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx). Para obter mais informações sobre downloads, confira os seguintes artigos:

- [Disponibilidade do produto](https://docs.microsoft.com/visualstudio/subscriptions/product-availability)
- [Baixar software](https://docs.microsoft.com/visualstudio/subscriptions/download-software)
