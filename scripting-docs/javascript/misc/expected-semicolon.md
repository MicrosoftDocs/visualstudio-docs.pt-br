# <a name="expected-"></a>';' esperado
Você tentou posicionar mais de uma instrução em uma linha, mas não as separava com pontos-e-vírgulas. Pontos e vírgulas são usados para encerrar instruções. Embora você possa inserir várias instruções em uma única linha, cada uma delas deve ser delimitada da próxima com um ponto-e-vírgula.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Marque o final de cada instrução com um ponto e vírgula.  
  
- Verifique se sua função chama o uso de parênteses corretamente.  
  
- Verifique se você não esqueceu os pontos-e-vírgulas dentro do cabeçalho de um loop **for** .
