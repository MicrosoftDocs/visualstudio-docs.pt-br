# <a name="c28208"></a>C28208
C28208 de aviso: A função \<function > foi definida anteriormente com uma lista de parâmetros diferente em \<file > (\<line >). Algumas ferramentas de análise produzirão resultados incorretos

 Esse aviso é relatado quando a definição conhecida de uma função não corresponde a outra ocorrência.
