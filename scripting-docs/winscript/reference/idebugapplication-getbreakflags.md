# <a name="idebugapplicationgetbreakflags"></a>IDebugApplication::GetBreakFlags
Retorna os sinalizadores de quebra atual para o aplicativo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetBreakFlags(  
   APPBREAKFLAGS*                   pabf,  
   IRemoteDebugApplicationThread**  pprdatSteppingThread  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pabf`  
 [out] Os sinalizadores de quebra atual para o aplicativo.  
  
 `pprdatSteppingThread`  
 [out] O thread em execução no momento.  
  
## <a name="return-value"></a>Valor de retorno  
 O método retorna um `HRESULT`. Os possíveis valores incluem, mas sem limitação, aqueles na tabela a seguir.  
  
|Valor|Descrição|  
|-----------|-----------------|  
|`S_OK`|O método foi bem-sucedido.|  
  
## <a name="remarks"></a>Comentários  
 Esse método retorna os sinalizadores de quebra atual para o aplicativo.
