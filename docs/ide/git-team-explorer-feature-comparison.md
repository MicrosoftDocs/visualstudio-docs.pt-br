# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Comparação lado a lado do git e do Team Explorer

Lançamos a primeira versão de uma nova experiência de git no Visual Studio 2019 [versão 16,8](/visualstudio/releases/2019/release-notes/). Essa nova experiência ajuda a reduzir a alternância de contexto com uma janela simples de **alterações do git** que inclui tarefas comuns de git. Ele também inclui uma janela de **repositório git** de toda a tela para operações git mais avançadas, como gerenciamento de Branch e navegação de repositório.

Se você estiver usando o Team Explorer, aqui está um guia passo a passo que explica como você pode usar a nova experiência de git.

> [!NOTE]
> A seção a seguir inclui capturas de tela ajustadas para caber nas colunas de uma tabela. Clique em cada captura de tela para exibir uma versão maior dele. (Se você estiver usando um dispositivo Touch, toque em cada captura de tela para exibir uma versão maior dele.)
