# <a name="c28285"></a>C28285
C28285 de aviso: Para função, erro de sintaxe no parâmetro

 A ferramenta de análise de código relata esse aviso quando um erro provável é encontrado no arquivo de modelo. Alguns erros de arquivo de origem também podem causar esses erros.
