# <a name="spy-views"></a>Exibições do Spy++
Spy + + exibe exibições de árvore de janelas, mensagens, processos e threads disponíveis em seu computador.

## <a name="in-this-section"></a>Nesta seção
 [Exibição do Windows](../debugger/windows-view.md) Exibe um modo de exibição de árvore das janelas e dos controles disponíveis.

 [Exibição de mensagens](../debugger/messages-view.md) Exibe o fluxo de mensagens associado a uma janela, processo ou thread.

 [Exibição de processos](../debugger/processes-view.md) Exibe um modo de exibição de árvore de processos ativos.

 [Exibição de threads](../debugger/threads-view.md) Exibe um modo de exibição de árvore de threads ativos.
