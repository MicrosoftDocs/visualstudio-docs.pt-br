# <a name="process-debug-manager"></a>Gerente de depuração de processos
O process debug manager (PDM) é um componente do Visual Studio que gerencia programas e processos, disponibilizando-os para o gerenciador de depuração de sessão e os motores de depuração.

 O PDM gerencia todos os processos que podem ser depurados. Para ser depurado, um programa deve ser registrado no PDM. Este registro é feito no momento em que o programa é lançado, seja por uma porta ou por um motor de depuração.
