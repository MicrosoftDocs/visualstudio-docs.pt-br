# <a name="the-connection-property-in-the-application-settings-file-is-missing-or-incorrect"></a>A propriedade de conexão no arquivo de configurações do aplicativo está ausente ou incorreta

A propriedade de conexão no arquivo de Configurações de Aplicativo está ausente ou incorreta. A cadeia de caracteres de conexão a *dbml* arquivo foi usado em seu lugar.

O *dbml* arquivo contém uma referência a uma cadeia de caracteres de conexão no arquivo de configurações do aplicativo não pode ser encontrado. Esta mensagem é informativa; a configuração de cadeia de caracteres de conexão será criada quando **Okey** é clicado.

Para responder a essa mensagem, selecione **Okey**. As informações de conexão que estão contidas na *dbml* arquivo é adicionado às configurações do aplicativo.
