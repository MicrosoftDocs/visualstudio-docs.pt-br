# <a name="iactivescriptstringcompare-interface"></a>Interface IActiveScriptStringCompare
A interface `IActiveScriptCompare` é usada para definir o método de comparação de cadeia de caracteres para o mecanismo de script.  
  
## <a name="methods-in-vtable-order"></a>Métodos em ordem vtable  
  
|Método|Descrição|  
|------------|-----------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Define o método de comparação de cadeia de caracteres para o mecanismo de script.|
