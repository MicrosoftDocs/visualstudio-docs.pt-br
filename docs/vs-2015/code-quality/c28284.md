# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28284: para símbolo, predicados atualmente não há suporte para símbolos não função  
  
 Este aviso indica que uma anotação condicional (predicado, `__drv_when`) foi encontrado em algo diferente de uma função.
