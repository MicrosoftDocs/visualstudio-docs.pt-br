# <a name="ca0504"></a>CA0504
Diretório de regra foi ignorado porque não existe

 Aviso CA504 é gerado quando um diretório especificado na **CodeAnalysisRuleDirectories** não foi possível encontrar a propriedade em um arquivo de configuração do projeto ou compilação. Para resolver esse problema, insira um caminho de diretório válido.
