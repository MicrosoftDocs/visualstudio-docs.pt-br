# <a name="image-service-tools"></a>Ferramentas de serviço de imagem
O SDK do VS inclui várias ferramentas para ajudar os construtores de extensão a usar o serviço Visual Studio imagem.

- A [Manifest from Resources](../../extensibility/internals/manifest-from-resources.md) de dados utiliza uma lista de recursos de imagem (arquivos .png ou .xaml) e gera um arquivo de manifesto de imagem.

- A [Manifest to Code](../../extensibility/internals/manifest-to-code.md) usa um arquivo de manifesto de imagem e gera um arquivo wrapper a ser usado em arquivos C++, C#, VB ou .vsct.

- O [Visualizador da Biblioteca de](../../extensibility/internals/image-library-viewer.md) Imagens pode carregar, editar e pesquisar manifestos de imagem.
