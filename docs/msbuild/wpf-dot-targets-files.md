# <a name="wpf-targets-files"></a>Arquivos .targets do WPF

O Windows Presentation Foundation (WPF) estende o MSBuild adicionando um conjunto de tarefas específicas do WPF combinadas em um arquivo especial *.targets,* *microsoft.WinFX.targets*. Este arquivo combina o conjunto de tarefas mSBuild necessárias para construir um projeto MSBuild no WPF.
