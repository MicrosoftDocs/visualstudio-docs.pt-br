# <a name="microsoft-tech-community-forum-support-in-visual-studio-subscriptions"></a>Suporte ao fórum do Microsoft Tech Community em assinaturas do Visual Studio
Conecte-se e entre em contato com usuários e especialistas técnicos da Microsoft sobre práticas recomendadas, arquitetura de design e as atualizações mais recentes.


## <a name="access-the-microsoft-tech-community"></a>Acesse a comunidade de tecnologia da Microsoft 
O acesso ao fórum da Comunidade é simples.  

1. Entrar no <https://my.visualstudio.com/benefits>
0. Localize o bloco **Microsoft Tech Community** e clique em **obter ajuda**

    > [!div class="mx-imgBorder"]
    > ![Bloco da comunidade técnica da Microsoft](_img/vs-tech-community/vs-tech-community-tile.png "Clique em ' obter ajuda ' no bloco Microsoft Tech Community")

0. Se você não tiver registrado anteriormente para a Comunidade, clique em **introdução**
0. Na seção **etapa 1: configurar seu perfil** :
   - Selecione o idioma de preferência
   - Insira seu email de trabalho
   - Criar o nome do seu perfil 
0. Na seção intitulada **etapa 2: selecione as comunidades para ingressar**, você pode selecionar várias comunidades clicando no botão de **junção** listado no cartão de informações para cada um.  Você também pode usar a caixa de pesquisa na extrema direita para filtrar a lista de comunidades disponíveis.  Para ver a lista completa de 90 comunidades, clique no link **Exibir todos (90)** abaixo dos cartões de informações. 
0. Para a **etapa 3: código de conduta**, leia o código de conduta e, em seguida, clique na caixa de seleção para indicar sua aceitação do código de conduta e dos termos de serviço.
0. Clique em **registrar** para concluir a configuração do seu perfil.

Isso é tudo para ela.  Agora você está pronto para usar o portal da Comunidade.  A home page tem notícias sobre a Comunidade.  Para ver o conteúdo específico das comunidades que você ingressou, clique no menu suspenso de **hubs de comunidade** na parte superior da página e escolha uma comunidade na sua lista.  Você sempre pode adicionar comunidades à lista clicando em **procurar todos os hubs de comunidade** clicando no Hub de sua escolha e, em seguida, clicando no botão de **junção** . 

## <a name="eligibility"></a>Qualificação
A disponibilidade do benefício Comunidade Azure depende do nível de sua assinatura, conforme mostrado abaixo:

|                                          Nível de Assinatura                                           |     Canais      |    Benefício    | Renovável? |
|-------------------------------------------------------------------------------------------------------|-------------------|---------------|------------|
|                           Visual Studio Enterprise (padrão), NFR<sup>1</sup>                            | VL, Azure, Retail |   Disponível    |    Sim     |
|                           Assinatura Visual Studio Enterprise com o GitHub Enterprise                           | VL, Azure, Retail |   Disponível    |    Sim     |
|                          Visual Studio Professional (Standard)                          | VL, Azure, Retail |   Disponível    |    Sim     |
|                          Assinatura Visual Studio Professional com o GitHub Enterprise                          | VL, Azure, Retail |   Disponível    |    Sim     |
|                              Visual Studio Test Professional (Padrão)                               |    VL, Retail     |   Disponível    |    Sim     |
|                                       Plataformas MSDN (Padrão)                                       |    VL, Retail     |   Disponível    |    Sim     |
| Visual Studio Enterprise (nuvem mensal), Visual Studio Professional (nuvem mensal)|        Tudo        | Não disponível |     Não     |

<sup>1</sup>  *inclui: não para revenda (NFR), Microsoft FTE, profissional mais valioso (MVP), diretor regional (RD), parceiro do setor do Visual Studio (VSIP), BizSpark, imagine, Microsoft Certified Trainer (MCT)*

> [!NOTE]
> A Microsoft não oferece mais assinaturas anuais do Visual Studio Professional e do Visual Studio Enterprise nas Assinaturas na Nuvem. Não haverá nenhuma alteração na experiência dos clientes existentes nem na capacidade de renovar, aumentar, diminuir ou cancelar suas assinaturas. Novos clientes são incentivados a ir para [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) a fim de explorar diferentes opções para comprar o Visual Studio.

Não tem certeza de qual assinatura você está usando?  Conecte-se ao [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) para ver todas as assinaturas atribuídas ao seu endereço de email. Se não vir todas as suas assinaturas, talvez você tenha uma ou mais atribuídas a outro endereço de email.  Você precisará entrar com esse endereço de email para ver as assinaturas.
