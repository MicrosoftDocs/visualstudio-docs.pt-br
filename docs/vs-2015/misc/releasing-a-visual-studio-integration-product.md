# <a name="releasing-a-visual-studio-integration-product"></a>Liberando um produto de integração do Visual Studio
Os tópicos nesta seção mostram como liberar seu produto de integração do Visual Studio criando e instalando VSPackages e usando o teste de integração do Visual Studio (VSIT).  
  
## <a name="in-this-section"></a>Nesta seção  
 [Instalando VSPackages](../misc/installing-vspackages.md)  
 Discute como criar e instalar VSPackages e como dar suporte a usuários que estão executando várias versões do Visual Studio ao mesmo tempo.  
  
 [Utilitário RegPkg](../extensibility/internals/regpkg-utility.md)  
 Descreve o utilitário RegPkg, que é usado para registrar um VSPackage e prepará-lo para implantação.  
  
 [Guia do usuário e a instalação da edição de parceiro Premier](http://msdn.microsoft.com/en-us/8ee4dad7-95d3-4f2d-a8d4-3ba9a80ecae2)  
 Fornece aos parceiros VSIP com uma versão redistribuível do ambiente de desenvolvimento integrado (IDE) do Visual Studio para que eles podem adicionar suas próprias ferramentas de desenvolvimento e linguagens de computador.
