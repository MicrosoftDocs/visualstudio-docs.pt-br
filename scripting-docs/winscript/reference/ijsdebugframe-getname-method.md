# <a name="ijsdebugframegetname-method"></a>Método IJsDebugFrame::GetName
Obtém o nome amigável do quadro de pilhas.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pName`  
 fora O nome do quadro de pilha.  
  
## <a name="return-value"></a>Valor retornado  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jscript9diag. h
