# <a name="command-line-switches-visual-studio-sdk"></a>Switches de linha de comando (Visual Studio SDK)
*Devenv.exe*, o executável [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] que inicia o ambiente de desenvolvimento integrado (IDE), suporta switches de linha de comando especificamente para o desenvolvimento de VSPackages. VsPackages também podem disponibilizar seus próprios switches na linha de comando Devenv. Os tópicos a seguir fornecem informações sobre como usar switches de linha de comando no IDE.

## <a name="in-this-section"></a>Nesta seção
- [Switches de linha de comando Devenv para desenvolvimento de pacote supérdia VS](../extensibility/devenv-command-line-switches-for-vspackage-development.md) Descreve os switches de linha de comando Devenv que você pode usar para simplificar o desenvolvimento, a depuração e a implantação do VSPackage.

- [Adicionar interruptores de linha de comando](../extensibility/adding-command-line-switches.md) Descreve como implementar, procurar e ler switches de linha de comando inseridos pelo usuário para o seu pacote.

## <a name="related-sections"></a>Seções relacionadas
- [Interruptores de linha de comando Devenv](../ide/reference/devenv-command-line-switches.md) Explica como o Devenv permite definir várias opções para o IDE e como você pode compilar, construir e depurar projetos da linha de comando. Além disso, há informações sobre a sintaxe dos switches Devenv e uma tabela de switches.
