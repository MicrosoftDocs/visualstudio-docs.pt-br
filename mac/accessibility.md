# <a name="accessibility"></a>Acessibilidade

Há uma série de ferramentas de acessibilidade e recursos incorporados no macOS que o ajudarão a usar com o Visual Studio para Mac.  Além destes, o Visual Studio for Mac também possui os seguintes recursos para torná-lo mais acessível para pessoas de habilidades variadas:

* Ampliação de texto do editor
* Ampliação do texto do bloco
* Personalização de tema de cor do editor de código
* Personalização de atalhos de teclado
* Navegação de teclado

> [!NOTE]
> Este guia refere-se ao Visual Studio for Mac. Para o Visual Studio no Windows, consulte [recursos de acessibilidade do Visual Studio](/visualstudio/ide/reference/accessibility-features-of-visual-studio).
