# <a name="using-voiceover-with-visual-studio-for-mac"></a>Usando voiceover com visual studio para Mac

Os usuários que necessitarem de um leitor de tela se beneficiarão do macOS VoiceOver. Visual Studio for Mac é acessível usando comandos comum VoiceOver.

## <a name="how-to-enable-and-use-voiceover"></a>Como: Ativar e usar o VoiceOver

Para ativar ou desativar a prensa VoiceOver **&#8984;F5**

Os comandos VoiceOver aparecem neste guia como ** _chave_ VO,** pelo qual **vo** refere-se ao modificador definido no aplicativo **VoiceOver Utility.** O modificador padrão **é o "** Por exemplo, dependendo do modificador VoiceOver, **VO M** significará **"M".** As teclas do cursor serão referidas como **➤ → ↑ ◗**.

## <a name="useful-voiceover-commands"></a>Comandos úteis voiceOver

Aqui está uma lista de alguns comandos comum de VoiceOver para Visual Studio para Mac:

|Recurso|Descrição|Atalho|
|-------|-----------|--------|
|Navegar|Navegue por todos os elementos da UI|**VO→**, **VO.**|
|Navegar|Entrar ou sair de elementos de IU|**VO...** **VO⇧↑**|
|Interagir|Selecione ou interaja com o controle em foco|**VO**|
|Interagir|Interaja com a barra de menus Visual Studio para Mac|**VO M**|

> [!TIP]
> Para obter mais informações sobre o uso do VoiceOver e uma lista abrangente de comandos, consulte [o Guia de Início de Partida de Voz da Apple](https://support.apple.com/en-us/guide/voiceover-guide/welcome/web).
