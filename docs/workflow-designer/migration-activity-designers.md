# <a name="migration-activity-designers"></a>Designers de atividade de migração

Os designers de atividade de migração são usados para chamar o fluxo de trabalho 3.0/3.5 atividades de um fluxo de trabalho que tem como alvo o .NET Framework 4 ou posterior.

Os tópicos contidos nesta seção fornecem orientação sobre como usar os designers de atividade do Designer de fluxo de trabalho.
