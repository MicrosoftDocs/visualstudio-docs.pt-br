# <a name="troubleshoot-clickonce-deployments"></a>Solucionar problemas de implantações do ClickOnce
Este tópico ajuda você a diagnosticar e resolver os problemas mais comuns com [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] implantações.

 Na maioria dos casos, um [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] aplicativo será baixado para o computador de um usuário e executado sem problemas. No entanto, há alguns casos em que problemas de configuração do servidor Web ou do aplicativo podem causar problemas imprevistos.
