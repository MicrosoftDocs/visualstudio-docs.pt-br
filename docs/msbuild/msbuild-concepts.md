# <a name="msbuild-concepts"></a>Conceitos do MSBuild

O MSBuild fornece um esquema XML básico que você pode usar para controlar como a plataforma de compilação constrói software. Para especificar os componentes no build e como eles devem ser compilados, use essas quatro partes do MSBuild: propriedades, itens, tarefas e destinos.
