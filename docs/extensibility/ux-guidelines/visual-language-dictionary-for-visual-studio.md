# <a name="visual-language-dictionary"></a>Dicionário de linguagem visual

Visual Studio usa uma linguagem visual comum para ilustrar os conceitos que são os blocos de construção de sua iconografia e outras imagens. Ao criar novos ícones, use esse dicionário para identificar os conceitos que você deseja transmitir em sua própria imagem.

## <a name="general-guidelines"></a>Diretrizes gerais

- Esses conceitos visuais devem ser usados consistentemente com sua definição de dicionário. Caso contrário, você corre o risco de confundir os usuários, que esperam que elementos comuns tenham significados consistentes.

- Escolha apenas os conceitos mais relevantes ou importantes para uso em seus ícones, para que a imagem seja perfeita e fácil de distinguir.

- Use apenas um ou dois conceitos em cada imagem.

- Para obter mais estratégias para criar ícones, consulte [Design de ícone.](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign)
