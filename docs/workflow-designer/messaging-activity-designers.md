# <a name="messaging-activity-designers"></a>Designers de atividade do sistema de mensagens

Designers de atividade de mensagens são usados para criar e configurar as atividades de mensagem que enviam e recebem mensagens do Windows Communication Foundation (WCF) de dentro de um aplicativo do Windows Workflow Foundation (WF). Cinco atividades de mensagem foram introduzidas no .NET Framework 4. O Designer de fluxo de trabalho fornece dois designers de modelo que permitem que você gerencie o sistema de mensagens dentro de um fluxo de trabalho.

Os tópicos contidos nesta seção e listados na tabela a seguir fornecem orientação sobre como usar a atividade de Designer de fluxo de trabalho e os designers de modelo.

- <xref:System.ServiceModel.Activities.CorrelationScope>

- <xref:System.ServiceModel.Activities.Receive>

- <xref:System.ServiceModel.Activities.Send>

- <xref:System.ServiceModel.Activities.ReceiveReply>

- <xref:System.ServiceModel.Activities.SendReply>
