# <a name="debugging-in-visual-studio"></a>Depurando no Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]
O depurador do Visual Studio ajuda você a observar o comportamento em tempo de execução do seu programa e localizar problemas. O depurador funciona com todas as linguagens de programação do Visual Studio e com as bibliotecas associadas. Com o depurador, você pode interromper a execução do programa para examinar o código, avaliar e editar as variáveis, exibir registros, ver as instruções criadas pelo código-fonte e exibir o espaço de memória usado pelo aplicativo.  
  
 Para obter uma introdução rápida a depuração e diagnóstico no Visual Studio, consulte [depurar, analisar e diagnosticar](https://www.visualstudio.com/features/debugging-and-diagnostics-vs).  
  
-   [O que há de novo no depurador no Visual Studio 2015](../debugger/what’s-new-for-the-debugger-in-visual-studio-2015.md) descreve a nova funcionalidade no Visual Studio 2015.  
  
-   [Depurador do Windows](../debugger/debugger-windows.md)relaciona as janelas do depurador diferentes.  
  
-   [Noções básicas do depurador](../debugger/debugger-basics.md) aborda os conceitos básicos de como usar o depurador do Visual Studio. Os tópicos incluem noções básicas de depuração, controle de execução, anexação a um programa em execução, depuração just-in-time, início automático do depurador, despejos, pontos de interrupção, Inspeção de seu programa, tratamento de exceções, edição e continuação, e uso de expressões no depurador  
  
-   [Mais recursos de depuração](../debugger/more-debugging-features.md)descreve os diferentes recursos de depuração.  
  
-   [Depuração remota](../debugger/remote-debugging.md) descreve como configurar e usar a depuração remota.  
  
-   [Depurar aplicativos multi-threaded](../debugger/debug-multithreaded-applications-in-visual-studio.md) descreve como depurar aplicativos multithread.  
  
-   [IntelliTrace](../debugger/intellitrace.md) descreve o recurso IntelliTrace.  
  
-   [Depurando aplicativos](../debugger/debugging-applications.md) descreve como depurar tipos diferentes de aplicativos.  
  
-   [Referência de Interface do usuário de depuração](../debugger/debugging-user-interface-reference.md) fornece referência informações nas janelas e caixas de diálogo usadas ao depurar seu aplicativo. Esses tópicos também estão disponíveis quando você pressiona F1 enquanto tiver o foco em um elemento de interface do usuário depuração, como o **pontos de interrupção** janela.  
  
-   [SDK de acesso à Interface de depuração](../debugger/debug-interface-access/debug-interface-access-sdk.md) descreve o Microsoft Debug Interface acesso Software Development Kit (SDK do DIA). O DIA SDK fornece acesso a informações de depuração armazenadas em arquivos de banco de dados do programa (.pdb) gerados por ferramentas de pós-compilador da Microsoft.
