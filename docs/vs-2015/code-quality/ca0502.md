# <a name="ca0502"></a>CA0502

Idioma sem suporte.

Esse erro ocorre quando o usuário tentar analisar um projeto do ASP.NET que contém um ou mais páginas ou classes escritas em um idioma diferente do c# ou Visual Basic. Para corrigir esse erro, remova o arquivo do projeto ou desabilitar análise de código.
