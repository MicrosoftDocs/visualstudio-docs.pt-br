# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C1253: não é possível carregar o arquivo de modelo.  
  
 A ferramenta de análise de código relata esse aviso quando há um erro interno no arquivo de modelo, não no código que está sendo analisado.
