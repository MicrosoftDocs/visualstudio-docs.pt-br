# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28218: para o parâmetro de função, o parâmetro da anotação não corresponde ao encontrado no arquivo  
  
 As anotações na linha atual e na linha da mensagem são inconsistentes. Isso não deve ser possível se as macros padrão estiverem sendo usadas para anotações; Esse aviso não deve ser visto no uso normal.
