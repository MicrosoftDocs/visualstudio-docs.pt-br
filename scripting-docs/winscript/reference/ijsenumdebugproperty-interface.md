# <a name="ijsenumdebugproperty-interface"></a>Interface IJsEnumDebugProperty
Fornece operações de navegação de propriedade.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp
IJsEnumDebugProperty : public IUnknown;  
```  
  
## <a name="members"></a>Membros  
  
### <a name="public-methods"></a>Métodos Públicos  
  
|Name|Descrição|  
|----------|-----------------|  
|[Método IJsEnumDebugProperty::GetCount](../../winscript/reference/ijsenumdebugproperty-getcount-method.md)|Obtém o número de propriedades do objeto.|  
|[Método IJsEnumDebugProperty::Next](../../winscript/reference/ijsenumdebugproperty-next-method.md)|Lê as propriedades deste objeto.|
