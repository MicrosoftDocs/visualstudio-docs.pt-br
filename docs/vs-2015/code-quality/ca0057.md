# <a name="ca0057"></a>CA0057

As violações não podem ser mapeadas para o conjunto atual de destinos e regras.

Esse erro ocorrerá se FxCopCmd. exe for invocado manualmente com um arquivo de projeto ou um relatório XML/imported que tenha violações que não podem ser mapeadas para o conjunto atual de destinos e regras.

Ocorrerá somente se a análise for feita usando a versão de linha de comando da ferramenta.
