# <a name="c28213"></a>C28213
warning C28213: O `_Use_decl_annotations_` anotação deve ser usada para fazer referência, sem modificações, uma declaração prévia. \<why>

 Esse aviso é relatado quando uma declaração anterior é referenciada sem necessários `_Use_decl_annotations_` anotação.
