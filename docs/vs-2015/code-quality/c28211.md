# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28211: nome de contexto estático esperado para SAL_context  
  
 Esse aviso indica que o operando para a `_Static_context_` anotação deve ser o nome de um contexto definido por uma ferramenta. Esse aviso nunca deve ocorrer em uso normal.
