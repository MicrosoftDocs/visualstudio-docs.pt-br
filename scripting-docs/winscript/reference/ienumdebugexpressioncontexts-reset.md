# <a name="ienumdebugexpressioncontextsreset"></a>IEnumDebugExpressionContexts::Reset
Redefine uma sequência de enumeração para o início.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT Reset();  
```  
  
#### <a name="parameters"></a>Parâmetros  
 Esse método não usa nenhum parâmetro.  
  
## <a name="return-value"></a>Valor de retorno  
 O método retorna um `HRESULT`. Os possíveis valores incluem, mas sem limitação, aqueles na tabela a seguir.  
  
|Valor|Descrição|  
|-----------|-----------------|  
|`S_OK`|O método foi bem-sucedido.|  
  
## <a name="remarks"></a>Comentários  
 Este método redefine uma sequência de enumeração para o início.
