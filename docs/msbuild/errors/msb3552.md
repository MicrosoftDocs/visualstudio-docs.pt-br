# <a name="msb3552-resource-file-filename-cannot-be-found"></a>MSB3552: O arquivo de recurso 'filename' não pode ser encontrado

Esse erro ocorre quando um arquivo de recurso de solicitação não foi encontrado. Verifique o caminho e a ortografia do nome do arquivo. Esses arquivos geralmente são especificados como `EmbeddedResource` itens no arquivo de projeto.
