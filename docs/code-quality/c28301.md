# <a name="c28301"></a>C28301
C28301 de aviso: Não há anotações para a primeira declaração de \<function >. \<note1 > consulte \<filename > (\<line >). \<note2 >

 Esse aviso é relatado quando as anotações não foram encontradas na primeira declaração de uma determinada função.
