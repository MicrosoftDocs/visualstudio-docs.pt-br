# <a name="ca0503"></a>CA0503

A propriedade foi preterida. Use a propriedade substituta.

CA0503 de aviso será gerado quando uma propriedade de análise de código preterido contém um valor no arquivo de configuração de compilação ou projeto. Para resolver este aviso, use a propriedade especificada que a substitui.
