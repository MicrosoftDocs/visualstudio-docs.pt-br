# <a name="warning-c26814-use-constexpr-for-constants-whose-value-is-known-at-compile-time-con5"></a>Aviso C26814 use constexpr para constantes cujo valor é conhecido no momento da compilação. (Con. 5)
A variável const '% Variable% ' pode ser computada em tempo de compilação. Considere usar constexpr (con. 5)

## <a name="example"></a>Exemplo
```cpp
const int foo = 1234;  // C26814 reported here.
constexpr int getMagicNumber()
{
    return 42;
}

void bar()
{
    const int myval = 3; // C26814 reported here
    const int magicNumber = getMagicNumber(); // C26814 reported here.
}
```
