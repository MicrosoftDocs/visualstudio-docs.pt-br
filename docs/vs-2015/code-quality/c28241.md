# <a name="c28241"></a>C28241
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28241: A anotação para função no parâmetro não é reconhecida.  
  
 Um nome de anotação não reconhecido foi usado. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso típico.
