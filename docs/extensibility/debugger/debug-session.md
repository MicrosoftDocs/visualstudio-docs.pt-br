# <a name="debug-session"></a>Sessão de depuração
Em termos da arquitetura de depurador, uma *sessão*:

- É o conjunto de todos os processos, relacionados ou não relacionados, que foram lançados ou anexados a partir de uma única instância do IDE.

- É implementado por um gerenciador de depuração de sessão (SDM). Para obter mais informações, consulte [Session Debug Manager](../../extensibility/debugger/session-debug-manager.md).
