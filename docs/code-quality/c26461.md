# <a name="c26461-useconstpointerarguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  O argumento de ponteiro '% % do argumento' para a função '% % de função' pode ser marcado como um ponteiro para `const`. Ver [con.3 de diretrizes de núcleo do C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
