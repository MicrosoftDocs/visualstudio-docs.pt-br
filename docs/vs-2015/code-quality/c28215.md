# <a name="c28215"></a>C28215
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28215: o typefix não pode ser aplicado a um parâmetro que já tem um typefix  
  
 A aplicação de uma `__typefix` anotação a um parâmetro que já tem essa anotação é um erro. As `__typefix` anotações são usadas apenas em alguns casos especiais e esse aviso não deve ser visto no uso normal.
