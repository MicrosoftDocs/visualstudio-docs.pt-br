# <a name="c28204"></a>C28204
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28204: \<função >: Apenas uma dessa sobrecarga e um de cada \<filename > (\<linha >) são anotadas para \<paramname >: ambos ou nenhum deles deve ser anotado.  
  
 Esse aviso é relatado quando ocorre um erro nas anotações.
