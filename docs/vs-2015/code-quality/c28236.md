# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28236: a anotação não pode ser usada em uma expressão  
  
 Esse aviso indica que foi feita uma tentativa de usar uma função declarada para ser uma anotação em um contexto de expressão. Isso não deve ser possível se as macros padrão estiverem sendo usadas para anotações; Esse aviso não deve ser visto no uso típico.
