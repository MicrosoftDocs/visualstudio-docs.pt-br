# <a name="list-of-available-services"></a>Lista de serviços disponíveis

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]e o Visual Studio SDK suporta os seguintes serviços. Alguns pacotes oferecem seus próprios serviços que não estão listados aqui — por exemplo, os serviços de idioma não têm um único GUID de serviço. Você deve usar o nome do idioma para encontrar o GUID do serviço de idiomas no registro.

Use os GUIDs de serviço listados aqui ou obtidos de alguma outra fonte (por exemplo, serviços de idioma) para obter a interface primária ou interfaces mostradas com cada serviço.
