# <a name="c28214"></a>C28214
C28214 de aviso: Os nomes do parâmetro de atributo devem ser p1...p9

 Esse aviso indica que quando você constrói uma declaração de anotação, os nomes de parâmetro são limitados a P1... P9. Esse aviso nunca deve ocorrer em uso normal.
