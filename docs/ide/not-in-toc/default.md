# <a name="f1-help"></a>ajuda F1

Parece que não foi possível encontrar o que você está procurando e pedimos desculpas. Você pode nos ajudar a melhorar deixando comentários usando os botões na parte inferior desta página.

## <a name="cursor-position"></a>Posição do cursor

Se você estiver procurando ajuda F1 em uma palavra-chave ou nome de API, certifique-se de posicionar o cursor dentro ou logo à direita da palavra antes de pressionar **F1**. Se isso ainda não funcionar, selecione na lista de [links úteis](#useful-links) a seguir ou faça uma pesquisa em https://docs.microsoft.com .
