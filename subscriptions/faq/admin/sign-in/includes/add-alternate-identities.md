## <a name="what-does-it-mean-when-my-subscribers-add-alternate-identities"></a>O que significa a adição de identidades alternativas feita pelos assinantes?

Adicionar uma conta alternativa permite que os desenvolvedores acessem alguns dos benefícios de assinatura do Visual Studio (VS IDE, Azure DevOps e Azure) com uma identidade diferente da que a assinatura é atribuída. Uma ID alternativa não fornece uma cópia da assinatura para a outra conta; ele só fornece acesso aos dois benefícios com a conta alternativa. [Leia mais sobre as identidades da Assinatura do Visual Studio](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity).
