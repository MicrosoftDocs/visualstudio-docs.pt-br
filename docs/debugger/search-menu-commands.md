# <a name="search-menu-commands"></a>Comandos do menu Pesquisa
O **pesquisa** menu fornece comandos para pesquisar específicos do windows.

 **Encontrar janela/processo/Thread/mensagem** abre o **janela**/**processo**/**Thread** /  **MessageSearch** caixa de diálogo, em que você pode usar o **FinderTool** ou especificar critérios de pesquisa.

 **Localizar próximo** procura o modo de exibição atual para a próxima janela correspondente, processo, thread ou mensagem. Este item de menu (e o botão de barra de ferramentas relacionados) está disponíveis somente quando há um resultado de pesquisa válido que não é exclusivo.

 **Localizar anterior** procura o modo de exibição atual para a janela correspondente anterior, processo, thread ou mensagem. Este item de menu (e o botão de barra de ferramentas relacionados) está disponíveis somente quando há um resultado de pesquisa válido que não é exclusivo.
