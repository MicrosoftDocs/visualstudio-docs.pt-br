# <a name="c26497-useconstexprforfunction"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

Essa função de % de função poderá ser marcada como `constexpr` se a avaliação do tempo de compilação for desejada. Ver [diretrizes principais do C++ F.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr).
