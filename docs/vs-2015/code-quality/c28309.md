# <a name="c28309"></a>C28309
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso 28309: <parameter_name>: os operandos de anotação devem ser tipos de número inteiro/enum/ponteiro. Não há suporte para operandos nulos e operadores sobrecarregados de C++. Os floats são aproximados como inteiros. Tipos: \<typelist> .  
  
 Você tentou usar um void ou uma função em uma expressão de anotação, e a análise de código não pode tratá-lo.  Esse erro normalmente ocorre quando um `operator==` que é implementado como uma função é usado, mas outros casos também podem ocorrer. Examine os tipos em \<typelist> para obter dicas sobre o que está errado.
