# <a name="c28283"></a>C28283
aviso C28283: for Symbol, a especificação de tamanho especificada ainda não tem suporte

 O aviso indica que a anotação "Sentinela" de especificação de tamanho recebeu um valor diferente de zero. Essencialmente, o chamador tentou dizer que a cadeia de caracteres foi encerrada por um caractere diferente de zero binário.
