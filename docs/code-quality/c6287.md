# <a name="c6287"></a>C6287
Aviso C6287: código redundante: as subexpressãos esquerda e direita são idênticas

 Esse aviso indica que um elemento redundante foi detectado em uma expressão.

 É difícil avaliar a gravidade desse problema sem examinar o código. Um teste duplicado por si só é inofensivo, mas as consequências de excluir o segundo teste podem ser graves. O código deve ser inspecionado para garantir que um teste não foi omitido.

## <a name="example"></a>Exemplo
 O código a seguir gera este aviso:

```
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 O código a seguir mostra várias maneiras de corrigir este aviso:

```
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
