# <a name="expected--javascript"></a>')' esperado (JavaScript)
Você tentou colocar uma expressão dentro de um conjunto de parênteses, mas não incluiu o parêntese de fechamento. Alguma expressão deve ser colocado dentro de um conjunto de abertura e fechamento de parênteses. Observe o uso de parênteses no exemplo a seguir.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione parênteses à direita para a expressão de avaliação.
