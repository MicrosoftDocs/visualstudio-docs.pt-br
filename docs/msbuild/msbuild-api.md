# <a name="use-the-msbuild-api"></a>Usar a API do MSBuild

O MSBuild fornece uma superfície de API pública para que seu programa possa executar builds e inspecionar projetos.

A documentação da API do MSBuild pode ser encontrada em [Namespaces do Microsoft.Build](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx).
