# <a name="debug-store-apps-in-visual-studio"></a>Depurar aplicativos da Store no Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

O depurador do Visual Studio permite controlar a execução de um programa e examinar seu estado. Você pode usar o depurador para encontrar a causa de defeitos em seu aplicativo da Windows Store e entender exatamente como o aplicativo funciona. Quando você suspende (pausa) a execução no depurador, o Visual Studio exibe o arquivo de origem que contém o código de execução e destaca a instrução em execução. Você pode ver os valores das variáveis​​, a pilha de chamadas de funções em execução e outros aspectos do estado do seu programa. Você pode continuar a executar (percorrer) no programa uma instrução de cada vez para ver como as instruções alteram os valores do programa. Em aplicativos escritos com JavaScript, você pode examinar e manipular o DOM da página.
