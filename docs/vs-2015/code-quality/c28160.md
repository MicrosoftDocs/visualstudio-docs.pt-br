# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28160: anotação de erro  
  
 Esse aviso é relatado quando uma `__drv_error` anotação é encontrada. Essa anotação é usada para sinalizar as práticas de codificação que devem ser corrigidas e pode ser usada com uma `__drv_when` anotação para indicar combinações específicas de parâmetros.
