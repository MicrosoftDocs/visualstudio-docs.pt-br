# <a name="c28160"></a>C28160
Aviso C28160: Anotação de erro

 Esse aviso é relatado quando uma `__drv_error` anotação foi encontrada. Essa anotação é usada para sinalizar práticas de codificação que devem ser corrigidas e podem ser usadas com um `__drv_when` anotação para indicar combinações específicas de parâmetros.
