# <a name="script_e_recorded-error-code"></a>Código de erro SCRIPT_E_RECORDED
Foi passado um erro entre o mecanismo de script e o host. O host precisa passar o código de erro para o chamador.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
