# <a name="idebugportsupplier2enumports"></a>IDebugPortSupplier2::EnumPorts
Recupera uma lista de todos os portos fornecidos por um fornecedor portuário.

## <a name="syntax"></a>Sintaxe

```cpp
HRESULT EnumPorts( 
   IEnumDebugPorts2** ppEnum
);
```

```csharp
int EnumPorts( 
   out IEnumDebugPorts2 ppEnum
);
```

## <a name="parameters"></a>parâmetros
`ppEnum`\
[fora] Retorna um objeto [IEnumDebugPorts2](../../../extensibility/debugger/reference/ienumdebugports2.md) contendo uma lista de portas fornecidas.

## <a name="return-value"></a>Valor retornado
 Se for `S_OK`bem sucedido, retorna; caso contrário, retorna um código de erro.
