## <a name="im-unable-to-activate-my-azure-devops-benefit"></a>Não consigo ativar meu benefício do Azure DevOps.

Como assinante do Visual Studio, você pode usar uma variedade de recursos no Azure DevOps. Você poderá usar essas mesmas funcionalidades em cada organização do Azure DevOps de que for membro, independentemente de ter criado a organização ou ter sido adicionado por outra pessoa.  

## <a name="sign-in"></a>Entrar
1. Entre no portal de [assinaturas](https://my.visualstudio.com/benefits) e clique no link introdução na parte inferior do bloco benefício do Azure DevOps.
1. Você será levado para o Azure DevOps, onde você deve concordar com os termos de serviço. 
1. Entre no Azure DevOps usando o email que você usa para entrar em sua assinatura do Visual Studio. Se preferir usar um email diferente para fazer logon, você poderá [Adicionar uma conta alternativa](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity) para vincular sua assinatura, permitindo que você acesse o Azure DevOps usando os dois emails.
