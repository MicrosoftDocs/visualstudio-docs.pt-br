# <a name="expected-3939"></a>Esperado &#39;: &#39;
Você tentou criar uma expressão usando o operador condicional Ternário, mas não inclui os dois-pontos entre o segundo e terceiro operandos. O operador condicional (três operandos) Ternário requer dois-pontos entre o segundo (true) e o terceiro (false) operandos.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Insira um vírgula entre o segundo e terceiro operandos.
