# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C6506: anotação inválida: \<nome > propriedade pode ser usada somente em valores de tipos de ponteiro ou matriz  
  
 Este aviso indica que uma propriedade é usada em um tipo diferente de tipos de ponteiro ou matriz. O acesso, Tainted e propriedades válidas podem ser usadas em todos os tipos de dados. Suportam a outras propriedades, como ValidBytesConst, ValidElementsConst, ElementSize e NullTerminted ponteiro, ponteiro para membro ou tipos de matriz. Para obter uma lista completa de propriedades e os tipos de dados com suporte, consulte [propriedades de anotação](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Exemplo  
 O código a seguir gera este aviso:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Para corrigir este aviso, use um ponteiro ou um tipo de matriz, conforme mostrado no código de exemplo a seguir:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
