# <a name="find-window-dialog-box"></a>Caixa de diálogo Localizar Janela
Use essa caixa de diálogo para selecionar uma janela e exibir suas propriedades ou mensagens. Para exibir essa caixa de diálogo, escolha **localizar janela** no menu do **Spy** .

 As seguintes configurações estão disponíveis:

 **Ferramenta de localizador** A ferramenta arrastar e soltar usada para selecionar uma janela.

 **Ocultar Spy + +** Remove a janela principal do Spy + + e deixa a caixa de diálogo **localizar janela** visível na parte superior dos outros aplicativos.

 **Identificador** do A ID exclusiva da janela a ser localizada.

 **Legenda** O texto da legenda da janela.

 **Classe** do A classe da janela.

 **Estilo** do O estilo da janela.

 **Rect** O retângulo delimitador para a janela.

 **Mostrar propriedades ou mensagens** Determina se pressionar OK exibirá a [caixa de diálogo Propriedades da janela](../debugger/window-properties-dialog-box.md) ou abrirá uma [exibição de mensagens](../debugger/messages-view.md).
