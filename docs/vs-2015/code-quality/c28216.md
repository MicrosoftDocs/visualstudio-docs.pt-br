# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

aviso C28216: a \_ \_ anotação Check_return só se aplica a pós-condições para o parâmetro de função específico.  
  
 A `_Check_return_` anotação foi aplicada em um contexto diferente de `__post` ; ela pode precisar de um `__post` `__drv_out` modificador (ou) ou pode ser colocada incorretamente.
