# <a name="ca0055"></a>CA0055
Não foi possível carregar o\<arquivo: caminho >

A ferramenta de análise de código não pôde carregar o arquivo de destino que foi especificado para análise. Isso ocorrerá se o arquivo não for encontrado ou se o acesso ao arquivo tiver sido negado. Normalmente, isso ocorre se o usuário optou por substituir a propriedade de assembly de entrada no Visual Studio ou se o FxCopCmd. exe foi invocado manualmente.
