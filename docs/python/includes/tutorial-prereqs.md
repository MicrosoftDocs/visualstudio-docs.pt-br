## <a name="prerequisites"></a>Pré-requisitos

- Visual Studio 2017 com a carga de trabalho do Python instalada. Para obter instruções, confira [Como trabalhar com o Python no Visual Studio – Etapa 0](../tutorial-working-with-python-in-visual-studio-step-00-installation.md).

Você também pode usar uma versão anterior do Visual Studio com as Ferramentas Python para Visual Studio instaladas. Confira [Instalar o suporte para Python no Visual Studio](../installing-python-support-in-visual-studio.md). Alguns recursos podem não estar disponíveis em versões mais antigas, conforme explicado na [Matriz de recursos](../overview-of-python-tools-for-visual-studio.md#features-matrix).
